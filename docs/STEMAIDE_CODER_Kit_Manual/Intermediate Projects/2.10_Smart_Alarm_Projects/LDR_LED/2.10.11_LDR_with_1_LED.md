# Project 2.9.1: SMART STREET LIGHT (1 LED)

| **Description** |This project demonstrates how to build an automatic smart street light system using an LDR (Light Dependent Resistor) and an LED. The Arduino reads the surrounding light intensity through the LDR sensor and automatically switches the LED ON when it becomes dark and OFF when sufficient light is detected. Through this project, you will learn how sensors can be used with microcontrollers to create automated lighting systems and understand the basic principles behind smart lighting technology.  |
|------------------|----------------------------------------------------------------|
| **Use case**     |Automatic street lighting systems, smart home lighting, security lights, outdoor pathway lighting, energy-saving lighting solutions, and any application where lights need to respond automatically to changes in surrounding light conditions. |
