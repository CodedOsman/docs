**Student & Teacher Manual**

**1. Project Overview**

First-Person View (FPV) flying transforms the pilot from a remote observer into a virtual cockpit occupant. By adding a camera, video transmitter, and receiving goggles to the quadcopter from Project 10, students experience flight from the aircraft's perspective — the same technology used in commercial drone inspections, search-and-rescue operations, and competitive FPV racing. This project also introduces the mandatory spotter role, mirroring real UAV regulations that require a visual observer during FPV flights.

**Learning Objectives**

* Install a camera, video transmitter, and antenna on the Project 10 quadcopter
* Explain the FPV signal chain: camera → VTX → antenna → goggles, and the role of latency
* Select and match the correct frequency channel on the VTX and goggles
* Describe the legal and safety requirement for a spotter during FPV flight
* Complete a 30-second stable hover using FPV view only, without direct visual contact with the drone
* Fly a straight 20-metre leg and pass through a 1 m gate using FPV only

**3. Build Steps & Assembly**

**Lesson 1 – Camera & VTX Installation**

|  |  |
| --- | --- |
| **STEP 1** | **Camera Mounting** |
|  | * Select the camera mounting position: forward-facing on the front frame plate, above the propeller plane * Tilt the camera 5–10° upward from horizontal — this shifts the horizon into the upper third of the FPV image, giving a better forward view at speed * Secure the camera with the supplied mount and zip ties — confirm it cannot rotate under vibration * Check: propeller tips must not appear in the camera frame at any stick position |

|  |  |
| --- | --- |
| **STEP 2** | **Video Transmitter Wiring** |
|  | * Identify the VTX power input: typically 5V or direct 3S battery voltage — check Kit 11 datasheet * Connect VTX power cable to the correct pad on the Power Distribution Board or flight controller 9V/5V pad * Connect camera video signal wire (yellow) → VTX video input * Connect GND from camera and VTX to a common ground * Do NOT power on yet — complete antenna installation first |

|  |  |
| --- | --- |
| **STEP 3** | **Antenna Mounting** |
|  | * Thread the VTX antenna through the top frame plate or mount it on a riser * Antenna must be vertical and pointing upward — horizontal antennas reduce range by up to 70% * Ensure the antenna is at least 3 cm away from any carbon fibre frame arm — carbon absorbs RF * Tighten SMA connector finger-tight only — do not use tools; SMA connectors can be damaged by over-tightening |

|  |  |
| --- | --- |
| **STEP 4** | **Goggle Setup & Channel Matching** |
|  | * Power on the goggles (battery or USB) without powering the drone * Navigate to the channel scan function in the goggles menu * Power on the drone (teacher connects LiPo); confirm VTX LED indicates correct power state * Run the channel scan on the goggles — the VTX channel should appear as the strongest signal * Select the matching channel; confirm clear image in goggles * Record the band and channel number in your logbook for future reference |

**Lesson 2 – Ground Systems Test**

|  |  |
| --- | --- |
| **STEP 5** | **Image Quality Check** |
|  | * Put on goggles; teacher holds the drone at arm's length and tilts in all directions * Student in goggles calls out what they see: 'nose up, tilting left, ground visible' * Confirm camera angle: horizon should be visible; propellers should not intrude into the frame * Check for jello (vibration-induced wobble) — if present, add foam vibration damping under camera mount * Walk 50 m away from the drone with goggles on; image should remain clear |

**Lessons 3–4 – Spotter Briefing & First FPV Flights**

|  |  |
| --- | --- |
| **STEP 6** | **Spotter Role Briefing** |
|  | * Assign one spotter per pilot — this is mandatory, not optional * Spotter responsibilities: maintain visual contact with the drone at all times; call out hazards; ready to call ABORT * Communication signals: 'CLEAR' = safe to fly; 'ABORT' = land immediately; 'HOLD' = hover and wait * Spotter stands beside (not behind) the pilot to watch the drone, not the pilot * The spotter has equal authority to the pilot to call an abort — this must be agreed and respected before flying |

|  |  |
| --- | --- |
| **STEP 7** | **FPV Hover – Goggles On** |
|  | * Pilot sits on a chair; puts on goggles; confirms clear image before arming * Teacher arms the drone and increases throttle to hover; hands transmitter to pilot * Pilot maintains hover for 30 seconds using FPV view only — no direct visual contact * Focus: maintain altitude and position; make small, gentle corrections * After 30 seconds, teacher takes the transmitter and lands; pilot reviews: what was hardest about the FPV view? |

|  |  |
| --- | --- |
| **STEP 8** | **FPV Forward Flight & Gate Pass** |
|  | * Set up a 1 m × 1 m gate frame 15 m from the takeoff point * Pilot arms drone in goggles; hovers; then flies slowly forward toward the gate * First attempt: fly through the gate at low speed (< 2 m/s) * Success: full pass without touching the gate frame * Repeat 3 times; on final attempt, pilot chooses their own approach speed * Complete all entries in the Test Verification Table (Section 6) |

**Lessons 5–6 – FPV Circuit & Competition**

|  |  |
| --- | --- |
| **STEP 9** | **FPV Circuit Flight** |
|  | * Set up a simple 3-gate circuit: Gate A (straight), Gate B (45° right turn), Gate C (45° left turn back to start) * Each pilot completes 3 laps; spotter calls the time * Scoring: fastest clean lap (no gate touches); 5-second penalty per gate touch * Award: fastest pilot, smoothest pilot (teacher judged), most improved |

**4. Power & Safety Notes**

|  |
| --- |
| **⚠ Safety Requirements**  Power: LiPo 3S battery. Teacher connects and disconnects only.  Spotter: MANDATORY for ALL FPV flights. No FPV flight without a briefed spotter.  Goggles: Pilot wearing goggles CANNOT see the real environment — spotter is their eyes.  GCAA regulation: Ghana Civil Aviation Authority requires a visual observer for all UAV FPV operations.  Flight zone: Minimum 30 m × 30 m outdoor area; no bystanders within 10 m of the course.  VTX power: Use 25 mW for ground tests and practice flights. Only increase outdoors for range tests.  VTX heat: Never power VTX without antenna attached — it will overheat and burn out within seconds. |

**5. Engineering Principles**

**FPV System Components**

| **Component** | **Role** | **Key Spec** | **Notes** |
| --- | --- | --- | --- |
| **FPV Camera** | Captures live video | 600–1000 TVL or HD | Wide-angle lens; 120–160° FOV preferred |
| **Video Transmitter (VTX)** | Broadcasts video wirelessly | 5.8 GHz; 25–200 mW | Lower power = less heat; check local regulations |
| **Antenna (VTX)** | Transmits signal to goggles | Circular polarised (cloverleaf) | Mount vertically; clear of frame arms |
| **FPV Goggles / Monitor** | Receives and displays video | Matched to 5.8 GHz | Diversity receiver preferred for range |
| **Spotter** | Eyes-on safety observer | Human | Mandatory during all FPV flights – not a component but a role |

**Frequency & Channel Reference**

| **Band** | **Channels** | **Frequencies (MHz)** | **Notes** |
| --- | --- | --- | --- |
| **Raceband (R)** | R1–R8 | 5658–5917 | Most popular for racing and freestyle |
| **Band A** | A1–A8 | 5865–5945 | Fatshark default; good for beginners |
| **Band B** | B1–B8 | 5733–5840 | Often used for second pilot in same area |
| **Band E** | E1–E8 | 5705–5885 | Avoid E4 and E5 – close to Wi-Fi channels |

|  |
| --- |
| **Latency – Why It Matters for FPV**  Latency is the delay between what the camera sees and what the pilot sees in the goggles.  Analogue 5.8 GHz systems have < 40 ms latency — fast enough to feel real-time.  Digital HD systems (DJI, HDZero, Walksnail) have 20–50 ms latency — comparable to analogue.  High latency (> 100 ms) makes the drone feel sluggish and hard to control — the pilot reacts to what already happened.  This is why commercial FPV systems are engineered to minimise latency, not maximise image quality. |

**Spatial Awareness in FPV**

* Without peripheral vision, depth perception is reduced — distances feel different through goggles
* Building a mental 3D map of the environment is the core skill in FPV flying
* Looking toward a gate before flying toward it — called 'spotting the gate' — is used by all skilled FPV pilots
* When disoriented: reduce throttle slowly to land; do not make sudden inputs

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **No video in goggles** | Wrong channel or band | Scan all channels in goggles; match to VTX channel setting |
| **Fuzzy or noisy image** | VTX on wrong channel or interference | Change VTX to Raceband R1; rescan goggles |
| **Video breaks up at 10 m** | VTX antenna not connected or lying flat | Check SMA connector; mount antenna vertically |
| **Spatial disorientation (FPV)** | First time wearing FPV goggles in flight | Stop and hover; look at drone through goggles slowly; only fly forward when confident |
| **Propellers in video frame** | Camera angle too low | Tilt camera slightly upward (5–10°); recheck in goggles |
| **VTX very hot** | Running at high power indoors | Reduce VTX to 25 mW for indoor ground tests; increase outdoors |

**9. Upgrade & Extension Ideas**

* HD FPV System – Replace analogue VTX with a DJI O3 or HDZero digital system; compare image quality and latency
* DVR Recording – Enable the goggle's built-in DVR; review your flight footage; identify technique improvements
* OSD (On-Screen Display) – Add battery voltage, flight time, and GPS coordinates overlaid on the FPV image
* Long-Range Antenna – Upgrade to a helical or patch antenna; test maximum video range in an open field
* FPV Race League – Set up a 6-gate course; run a timed club competition; award points across 3 race days

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* Assign spotter roles carefully — a good spotter is as important as a good pilot; rotate both roles across the class
* Run the ground image test (Step 5) before any flight — it catches camera angle and wiring problems before they become airborne problems
* Never allow a pilot to fly FPV alone — the spotter rule is non-negotiable and reflects real GCAA UAV regulation
* For Lesson 3, start with the pilot sitting down for the hover — removing the need to balance reduces cognitive load

**Differentiation Strategies**

* Support – FPV hover only (30 s); no gate challenge; spotter-assisted flight with teacher guidance throughout
* Core – Hover, forward flight, and gate pass; FPV circuit; full scoring
* Extension – HD system comparison; DVR review; OSD overlay; race league organisation

**Assessment Suggestions**

* System installation: Camera angle correct; VTX channel matched; antenna vertical
* Spotter brief: Student can explain spotter role and three communication signals correctly
* FPV hover: 30-second stable hover on FPV view demonstrated to teacher
* Gate pass: At least one clean gate pass recorded and confirmed by spotter
