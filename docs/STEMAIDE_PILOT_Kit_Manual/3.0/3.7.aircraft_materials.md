**Student & Teacher Manual**

**1. Project Overview**

The choice of structural material is one of the most consequential decisions in aircraft design. Every gram of unnecessary weight costs fuel, range, and payload capacity — yet every structure must be strong enough to carry the loads it will encounter. This project puts five aviation materials in students' hands: aluminium, carbon fibre, fibreglass, balsa, and foam. Through systematic bending stiffness and tensile strength tests, students generate their own data — and discover why carbon fibre has become the dominant material in modern aircraft design.

**Learning Objectives**

* Identify and describe the five primary structural materials used in aviation: aluminium, carbon fibre, fibreglass, balsa, and foam
* Perform a bending stiffness test: apply known loads to a simply supported beam; measure deflection
* Perform a tensile strength test: apply increasing load until failure; record failure load
* Calculate specific strength (strength ÷ density) for each material and explain its significance
* Recommend the best material for each of four aircraft structural zones based on test data
* Explain why carbon fibre composite has replaced aluminium as the primary structural material on modern aircraft

**3. Build Steps & Assembly**

**Lesson 1 – Bending Stiffness Test**

|  |  |
| --- | --- |
| **STEP 1** | **Setup the Bending Test Rig** |
|  | * Place two pencils or blocks 15 cm apart on a flat table — these are the supports * Lay the test material strip across the supports, centred (7.5 cm overhang each side) * Hang the spring scale from the midpoint of the strip using a wire hook * Place a ruler vertically beside the midpoint to measure deflection * Zero the measurement: note the starting height at the midpoint with no load |

|  |  |
| --- | --- |
| **STEP 2** | **Bending Test Procedure** |
|  | * Add 50 g weight to the spring scale; wait 10 seconds; read the deflection at the midpoint * Add another 50 g; measure again; continue until deflection exceeds 20 mm or material fails * Record load (g) and deflection (mm) at each step * Calculate bending stiffness: K = Load / Deflection (g/mm) — higher K = stiffer material * Repeat for all 5 materials using fresh strips |

**Lesson 2 – Tensile Strength Test**

|  |  |
| --- | --- |
| **STEP 3** | **Setup the Tensile Test** |
|  | * Clamp one end of the material strip in a vice or secure it tightly to the table edge * Attach the spring scale to the free end using a loop of tape or a clamp * Slowly increase the pulling force by hand; read the spring scale * Record the maximum force just before the material fails (breaks or permanently deforms) * Record the material weight on the digital scale |

|  |  |
| --- | --- |
| **STEP 4** | **Calculate Specific Strength** |
|  | * Specific strength = Failure Load (g) ÷ Material Weight (g) * This measures how much load the material can carry per gram of its own weight * Plot a bar chart: Material (x-axis) vs. Specific Strength (y-axis) * Compare to published values in the Materials Reference Table (Section 5) |

**Lesson 3 – Analysis & Recommendations**

|  |  |
| --- | --- |

|  | * Using your test data, recommend the best material for each aircraft zone: * Wing spar (highest bending loads; low weight critical): \_\_\_\_\_\_\_ * Fuselage skin (moderate loads; smooth surface required): \_\_\_\_\_\_\_ * Landing gear (high impact loads; not weight-critical): \_\_\_\_\_\_\_ * Fairings and covers (low loads; smooth surface; easy to repair): \_\_\_\_\_\_\_ * Write a 2-sentence justification for each recommendation using specific test data |

| **⚠ Safety Notes**  Carbon fibre: Handle ONLY with protective gloves and safety goggles. Carbon fibre micro-filaments can penetrate skin and eyes. Never cut carbon fibre without eye protection.  Fiberglass: Wear gloves and a dust mask when handling fibreglass cloth — glass fibres irritate skin and lungs.  Failure tests: Stand clear when applying maximum loads — some materials fail suddenly and projectile fragments are possible.  Clean up all fibre fragments immediately after testing using damp paper towels. |

**5. Engineering Principles**

**Aviation Materials Reference Table**

| **Material** | **Density (g/cm³)** | **Tensile Strength (MPa)** | **Specific Strength (MPa·cm³/g)** | **Primary Aircraft Use** |
| --- | --- | --- | --- | --- |
| **Aluminium 6061** | 2.70 | 276 | 102 | Fuselage skin; wing ribs; frames |
| **Carbon Fibre (CFRP)** | 1.60 | 1500+ | 937+ | Spar caps; fuselage shells; rotor blades |
| **Fibreglass (GFRP)** | 2.00 | 440 | 220 | Fairings; doors; secondary structures |
| **Balsa Wood** | 0.16 | 20 | 125 | Model aircraft; core in sandwich panels |
| **EVA Foam** | 0.03–0.06 | 0.5–2 | 15–30 | Model aircraft; cushioning; packaging |
| **Steel (mild)** | 7.85 | 400 | 51 | Landing gear; engine mounts; fasteners |

|  |
| --- |
| **Why Specific Strength Matters More Than Absolute Strength**  A steel beam is stronger than a carbon fibre beam of the same size — but it is also 5× heavier.  In aircraft design, what matters is strength per kilogram of weight — specific strength.  Carbon fibre's specific strength is over 10× that of aluminium and over 18× that of steel.  This is why the Boeing 787 Dreamliner fuselage is 50% carbon fibre by weight — and burns 20% less fuel than the all-aluminium 767.  The trade-off: carbon fibre is expensive, brittle under impact, and difficult to inspect for internal damage. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Spring scale reading fluctuates** | Material stretching gradually | Wait 10 s before reading; take average of 3 readings |
| **Balsa fails before test starts** | Grain direction wrong | Cut balsa along the grain for tensile tests; use cross-grain only for shear tests |
| **Fibreglass cloth hard to grip** | Slippery surface | Fold a 2 cm tab at each end and clamp securely; use sandpaper on the grip area |
| **Foam compresses instead of failing** | EVA foam is highly deformable | Note the deformation rather than failure — foam is not a structural material; this is expected |
| **Deflection hard to measure accurately** | Eye-level variation | Set up the beam at table edge; use a ruler clamped vertically beneath the deflection point |

**9. Upgrade & Extension Ideas**

* Fatigue Test: apply repeated lower loads (50% of failure load) to the same aluminium strip 1000 times; check if failure load decreases — this is metal fatigue
* Sandwich Panel: laminate a foam core between two fibreglass skins; test bending stiffness compared to fibreglass alone — observe the dramatic increase
* Temperature Effect: heat an aluminium strip to 100°C in boiling water; test bending stiffness immediately — observe reduction in stiffness
* Cost Analysis: research market prices for each material per square metre; add cost per unit strength to your comparison

**10. Teacher Notes & Differentiation**

* The carbon fibre safety briefing (Step 1 of the test) is non-negotiable — spend 5 minutes on it before distributing any CFRP samples
* If carbon fibre samples are unavailable, demonstrate its properties using published data and video of aircraft wing bending tests
* Support – 3 materials only (balsa, foam, aluminium); bending test only; no specific strength calculation
* Core – All 5 materials; both tests; specific strength calculation; bar chart; material recommendation report
* Extension – Fatigue test; sandwich panel; temperature effect; cost analysis
