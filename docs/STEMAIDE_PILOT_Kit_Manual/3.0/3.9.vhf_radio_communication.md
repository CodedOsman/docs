**Student & Teacher Manual**

**1. Project Overview**

Radio communication is the primary means by which pilots and air traffic controllers coordinate the safe movement of aircraft. Every word, every number, and every pause in an aviation radio transmission follows a strict international standard defined by ICAO. This project gives students direct, immersive experience of aviation radiotelephony through structured role-play scenarios — practising the exact phraseology used at Kotoka International Airport and every other aerodrome in the world.

**Learning Objectives**

* Recite the complete ICAO phonetic alphabet (Alpha through Zulu) without reference material
* Format a correct ICAO callsign using the airline designator and flight number convention
* Perform four standard radiotelephony scenarios: taxi, takeoff, en-route position report, and approach/landing
* Correctly read back all ATC clearances using exact ICAO read-back requirements
* Initiate a MAYDAY emergency call with all required elements: call sign × 3, nature of emergency, intention
* Perform both the pilot and ATC controller role for at least one complete flight scenario

**3. Session Steps**

**Lesson 1 – Phonetic Alphabet & Basic Phraseology**

|  |  |
| --- | --- |
| **STEP 1** | **Phonetic Alphabet Drill** |
|  | * Distribute phonetic alphabet cards; give students 5 minutes to study * Teacher calls out a random letter; students call out the phonetic word * Speed round: teacher calls letters at 1-second intervals; class responds * Pairs test each other: one spells a 4-letter word phonetically; partner writes the letters * Standard: all 26 words from memory in under 60 seconds before proceeding |

|  |  |
| --- | --- |
| **STEP 2** | **Callsign & Number Format** |
|  | * ICAO callsign format: airline designator + flight number (e.g. Africa World Airlines 302 = 'AWA THREE ZERO TWO') * Numbers are spoken digit-by-digit: 302 = 'three zero two'; 21 = 'two one' * Altitude: '4500 feet' = 'four thousand five hundred feet'; 'FL120' = 'flight level one two zero' * Practice exercise: each student creates a fictional callsign and reads it aloud in ICAO format |

|  |  |
| --- | --- |
| **STEP 3** | **Basic Phraseology Study** |
|  | * Review the phraseology table (Section 5) for each of the 4 scenarios * Key phrases to memorise: 'Request', 'Roger', 'Wilco', 'Affirm', 'Negative', 'Standby', 'Go ahead' * Rule of radio communication: always begin with YOUR callsign, then the station you are calling * Read-back rule: all clearances (headings, altitudes, runway instructions) must be read back word-for-word |

**Lesson 2 – Scenario Role-Play**

|  |  |
| --- | --- |
| **STEP 4** | **Scenario 1 – Taxi & Departure** |
|  | * Student A = Pilot (Ghana Eagle 101); Student B = ATC (Accra Ground) * Pilot: 'Accra Ground, Ghana Eagle One Zero One, request taxi to runway Two One' * ATC: 'Ghana Eagle One Zero One, Accra Ground, taxi to holding point Runway Two One via Alpha' * Pilot reads back: 'Taxi holding point Runway Two One via Alpha, Ghana Eagle One Zero One' * Continue through takeoff clearance and departure; swap roles and repeat |

|  |  |
| --- | --- |
| **STEP 5** | **Scenarios 2–4 & MAYDAY** |
|  | * Scenario 2: En-route position report (pilot calls ATC with position, altitude, estimate for next point) * Scenario 3: Approach and landing (downwind, base, final calls; landing clearance; vacating runway) * Scenario 4: MAYDAY — pilot declares engine failure; ATC coordinates emergency services * For the MAYDAY: pilot says 'MAYDAY MAYDAY MAYDAY, Ghana Eagle One Zero One, engine failure, returning to Accra, 4500 feet, one POB' * ATC responds: acknowledges MAYDAY; all other traffic instructed to standby |

|  |  |
| --- | --- |
| **STEP 6** | **Assessment & Debrief** |
|  | * Teacher observes 2–3 pairs completing a full scenario; scores using the assessment rubric * Class debrief: what was hardest? What would you do differently? * Listen to a real ATC recording from liveatc.net (Accra frequency if available); identify phraseology from this lesson * Extension: visit the Kotoka ATC tower virtually or in person; listen to real transmissions on the public frequency |

**4. Safety Notes**

|  |
| --- |
| **Radio Frequency Regulations**  Real VHF aviation frequencies (118–137 MHz) are strictly regulated by the GCAA and ICAO.  Students must NEVER transmit on aviation frequencies — this is illegal and dangerous.  If handheld radios are used in this project, ensure they are set to a non-aviation frequency (e.g. FRS/GMRS) or used in simplex intercom mode.  The Kit 11 radios, if provided, are pre-configured for classroom simulation on approved frequencies. |

**5. Phraseology Reference & Scenarios**

**Standard Phraseology Reference**

| **Situation** | **Student (Pilot) Says** | **ATC Responds** | **Key Phrase** |
| --- | --- | --- | --- |
| **Request taxi** | [Callsign] request taxi to runway 21 | [Callsign] taxi to holding point runway 21 via taxiway Alpha | 'Roger', 'Wilco', 'Affirm' |
| **Takeoff clearance** | [Callsign] holding point runway 21, ready | [Callsign] wind 210 at 12, runway 21, cleared for takeoff | 'Cleared for takeoff' |
| **Position report** | [Callsign] passing [position] at [altitude] | Roger [callsign] | 'Passing', 'Flight level' |
| **Request landing** | [Callsign] 10 nm final runway 21 | [Callsign] wind calm, runway 21, cleared to land | 'Cleared to land' |
| **Emergency** | [Callsign] MAYDAY MAYDAY MAYDAY, [nature of emergency] | All stations stop transmitting; [callsign] state intentions | 'MAYDAY x3' |

**7. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Using casual language** | No exposure to ICAO phraseology | Print the phraseology card; insist on ICAO format; re-read the transmission correctly |
| **Forgetting callsign** | Not using the callsign consistently | Write the callsign on a card in front of the student; every transmission begins with it |
| **Incorrect read-back** | Read-back missing critical information | Teach the rule: read back all clearances, headings, altitudes, and runway instructions word-for-word |
| **Speaking too fast** | Nervousness; normal speech patterns | Slow down deliberately; speak at half normal speed; VHF radio voice is always measured |
| **Wrong ICAO word** | Alphabet not memorised yet | Practise the alphabet in pairs before any radio simulation begins |

**8. Upgrade & Extension Ideas**

* Live ATC Monitoring: listen to real Accra approach control on liveatc.net; identify each transmission's phraseology elements
* Full Flight Simulation: combine with the RC simulator (Project 4) — one student flies while another provides ATC instructions via radio
* ATIS Recording: write and record a simulated ATIS broadcast for Kotoka Airport using that day's weather station data from Project 36
* Emergency Coordination: add a third role (Emergency Coordinator); practice a full airport emergency response communication chain

**9. Teacher Notes & Differentiation**

* Play a real ATC recording at the start of Lesson 1 before any instruction — students are immediately intrigued by the rapid, precise language they are about to learn
* Laminate the phonetic alphabet cards so they can be used across multiple classes and years
* Support – Phonetic alphabet and taxi scenario only; scripts provided in full; no read-back requirement
* Core – All 4 scenarios; MAYDAY; read-backs; both pilot and ATC roles; assessment
* Extension – Live ATC monitoring; full flight simulation integration; ATIS recording
