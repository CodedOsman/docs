**Student & Teacher Manual**

**1. Project Overview**

Before GPS, every pilot navigated by dead reckoning: using only a compass, a clock, and a knowledge of speed and wind to fly from one point to another. Dead reckoning is still a foundational skill — GPS can fail, and a pilot who cannot navigate without it is a pilot who cannot be trusted with a real aircraft. In this project, students plan a three-leg cross-country flight across Ghana (Accra → Kumasi → Tamale → Accra) on a real aeronautical chart, then fly it in the RC simulator using heading and time only — no GPS overlay — before comparing their actual arrival position with their planned position.

**Learning Objectives**

* Define dead reckoning and explain why it remains a required skill despite the availability of GPS
* Use a scaled aeronautical chart of Ghana to measure track distances and true courses between three waypoints
* Apply wind correction angle and magnetic variation to calculate the magnetic heading to fly on each leg
* Estimate flight time for each leg using planned airspeed and leg distance
* Fly all three legs in the RC simulator using compass heading and timer only, with no GPS overlay
* Calculate the positional error at each waypoint and identify the largest source of error

**3. Session Steps**

**Lesson 1 – Flight Planning**

|  |  |
| --- | --- |
| **STEP 1** | **Introduce Dead Reckoning** |
|  | * Definition: Dead Reckoning (DR) = calculating current position from last known position + heading + speed + time * Formula: Distance = Speed × Time; rearranged for navigation: Time = Distance / Speed * Example: flying at 100 knots on a heading of 315° for 65 minutes = 108 nm traveled at 315° * Explain: GPS gives you position; DR gives you understanding — without understanding, GPS failure is catastrophic |

|  |  |
| --- | --- |
| **STEP 2** | **Chart Work – Measure Distances & Courses** |
|  | * Open the aeronautical chart of Ghana; locate DGAA (Accra), DGSI (Kumasi), DGLE (Tamale) * Using dividers and the chart scale bar, measure the direct distance for each leg in nautical miles * Using a protractor centred on the departure point, measure the true course for each leg (direction FROM departure TO destination, measured clockwise from North) * Record all values in the Flight Planning Worksheet (Section 6) |

|  |  |
| --- | --- |
| **STEP 3** | **Wind Correction & Magnetic Heading** |
|  | * Apply wind correction angle (WCA): if wind is from the left, add WCA to true course; from the right, subtract * For this exercise, use a fixed wind: 030°/15 knots * Apply magnetic variation for Ghana: approximately –3° (westerly variation) * Magnetic heading = True heading – Variation (for westerly) = True course + WCA + 3° * Estimate flight time: Time (min) = Distance (nm) / Groundspeed (knots) × 60 |

**Lesson 2 – Simulator Navigation Flight**

|  |  |
| --- | --- |
| **STEP 4** | **Simulator Setup** |
|  | * Load the RC simulator; select a fixed-wing aircraft similar to a training aircraft * Disable the GPS map overlay — navigation must be done by heading and timer only * Set the starting position to approximately 5.6°N, 0.2°W (Accra area) in the simulator * Take off and climb to 500 ft above terrain; set compass to Leg 1 magnetic heading |

|  |  |
| --- | --- |
| **STEP 5** | **Fly All Three Legs** |
|  | * Leg 1 (Accra → Kumasi): set heading; start timer; fly until elapsed time equals Leg 1 estimate * When timer reaches Leg 1 time: stop the clock; note your position — this is your estimated Kumasi * Without adjusting position, set heading for Leg 2 (Kumasi → Tamale); start timer again * Repeat for Leg 3 (Tamale → Accra) * Record all headings actually flown and all elapsed times in the Simulator Log (Section 6) |

**Lesson 3 – Analysis & GPS Comparison**

|  |  |
| --- | --- |
| **STEP 6** | **Position Error Analysis** |
|  | * At the end of Leg 3, enable the GPS map overlay; compare your actual position to DGAA * Measure the distance between your final position and Accra airport * Identify which leg had the largest error by reviewing your heading accuracy notes * Write a 2-paragraph analysis: what caused the error? What would you do differently? * Discussion: if you were flying a real aircraft, how would you check your position without GPS? (Visual landmarks, radio navigation, ATC position reports) |

**5. Engineering Principles**

**Dead Reckoning Navigation Concepts**

* True Course: the direction from departure to destination measured on a chart from True North
* Magnetic Variation: the difference between True North and Magnetic North at any given location; in Ghana, approximately 3° West (subtract 3° from true heading to get magnetic heading)
* Wind Correction Angle (WCA): the angle the aircraft must point into the wind to maintain the desired track over the ground
* Groundspeed: the aircraft's actual speed over the ground = True Airspeed ± wind component along the track

|  |
| --- |
| **Why Pilots Still Learn Dead Reckoning**  GPS satellites can be jammed, spoofed, or fail entirely — this has happened over conflict zones and been tested by adversaries.  ICAO requires all instrument-rated pilots to demonstrate dead reckoning proficiency.  The Ghana Civil Aviation Authority (GCAA) includes navigation theory in all pilot licence examinations.  A pilot who cannot navigate without GPS is dependent on a single point of failure — unacceptable in aviation where redundancy is a core safety principle. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Heading calculation wrong** | Magnetic variation not applied | Ghana variation is approx. -3° (westerly); subtract from true course to get magnetic heading |
| **Aircraft far off course after Leg 1** | Heading not held precisely in simulator | Practice holding a heading within ±2° for 30 seconds before starting the exercise |
| **Time estimate very inaccurate** | Groundspeed not correctly estimated | Check simulator airspeed indicator; use actual indicated speed for time calculations |
| **Waypoint missed entirely** | Wrong heading flown or timer not used | Return to the starting point; refly Leg 1 using a heading bug or compass reference |
| **Chart distances measured wrong** | Wrong scale used on chart | Confirm scale bar on the Ghana aviation chart; use dividers, not a ruler, for curved routes |

**9. Upgrade & Extension Ideas**

* Four-Leg Flight: extend to a 4-leg flight including a diversion: Accra → Kumasi → Sunyani → Tamale → Accra
* Radio Navigation: add a VOR fix at the midpoint of each leg using the simulator's VOR display to check position
* Real Chart vs Simulator: compare the simulator scenery to Google Maps at each waypoint; note differences
* GPS vs DR Race: two students fly the same route simultaneously — one uses GPS, one uses dead reckoning; compare arrival accuracy

**10. Teacher Notes & Differentiation**

* Print the Ghana aeronautical chart (VFR chart from SkyVector.com) at A3 size before Lesson 1 — having a real chart in hand is a powerful motivator
* For the simulator flight, have one student fly while a partner reads the heading and calls the timer — this mirrors how real aircraft with two pilots share workload
* Support – Single-leg exercise (Accra to Kumasi only); chart distances provided; focus on heading and timing
* Core – Full three-leg flight; chart work; wind correction; simulator navigation; error analysis
* Extension – Four-leg diversion; VOR fix; GPS vs DR race
