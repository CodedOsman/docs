**Student & Teacher Manual**

**1. Project Overview**

The obstacle course race combines precision, speed, and composure under pressure into a single timed challenge. Students must navigate four gates and a four-pylon slalom before executing a precision landing — all against the clock. Unlike free flight, the course imposes specific path constraints that force pilots to plan ahead, manage energy, and make rapid decisions. These are the same cognitive skills required of UAV inspection pilots navigating tight infrastructure environments.

**Learning Objectives**

* Set up a structured obstacle course with gates, slalom pylons, and a landing zone
* Fly the complete course sequence (Gate A → B → C → Slalom → Landing Zone) without missing elements
* Apply the look-ahead technique: identifying the next element before reaching the current one
* Complete a clean timed run in under 90 seconds
* Apply the penalty system correctly and record all three timed runs in the timing sheet
* Identify the highest-penalty element across all runs and describe how to improve it

**3. Session Steps**

**Lesson 1 – Course Setup & Walk-Through**

|  |  |
| --- | --- |
| **STEP 1** | **Course Layout** |
|  | * Select an outdoor area: minimum 30 m × 20 m, clear of obstacles and bystanders * Mark the course boundary with cones or tape — pilots must not exit this boundary * Place Gate A at the start: 1.5 m wide × 1 m tall; 5 m from the takeoff point * Place Gate B 10 m ahead of Gate A: 1.2 m wide; requires a 30° right turn from A * Place Gate C 8 m from Gate B: 1.2 m wide × 1.2 m tall; elevated 30 cm from ground * Place 4 slalom pylons in a line, 1 m apart, 8 m from Gate C * Place the 1 m landing zone 5 m beyond the last slalom pylon |

|  |  |
| --- | --- |
| **STEP 2** | **Course Walk-Through** |
|  | * All pilots walk the course on foot before flying * Each pilot narrates the route aloud: 'Take off; fly forward through Gate A; turn right toward Gate B...' * Identify the decision points: where to start turning for each gate; when to reduce speed for the slalom * Identify the look-ahead points: where to start looking at the next gate while still flying through the current one * Teacher confirms each pilot can describe the full course correctly before flying |

**Lesson 2 – Element-by-Element Practice**

|  |  |
| --- | --- |
| **STEP 3** | **Individual Element Practice** |
|  | * Practice Gate A 5 times before moving to Gate B * Criteria: centre of the drone passes through the centre of the gate; no touches * Once Gate A is clean, add Gate B; practice A → B 5 times * Add Gate C; practice A → B → C 5 times * Add slalom; practice C → Slalom 5 times * Only attempt the full course once all individual elements are clean |

|  |  |
| --- | --- |
| **STEP 4** | **Slalom Technique** |
|  | * Approach the slalom slowly (< 2 m/s) on first attempts * The correct path: fly just to the LEFT of pylon 1; just to the RIGHT of pylon 2; left of 3; right of 4 * Do not try to fly fast through the slalom until the weave path is consistent * Common error: looking at the pylon being passed; correct technique: look at the NEXT pylon * Once consistent at slow speed, increase to medium speed for timed runs |

**Lessons 3–4 – Timed Runs & Competition**

|  |  |
| --- | --- |
| **STEP 5** | **First Timed Run (Practice)** |
|  | * Fly the complete course; timekeeper starts on takeoff; stops on landing inside zone * Any penalties are added to the raw time * No pressure: focus on completing the course cleanly, not on time * Debrief: which element had the most penalties? That is the practice priority |

|  |  |
| --- | --- |
| **STEP 6** | **Scored Timed Runs** |
|  | * Each pilot completes 3 timed runs * Best time (including penalties) is the competition score * A dedicated spotter stands at each gate to confirm touches and call penalties * Gate spotter raises one hand for a touch; two hands for a missed gate * Record all times and penalties in the Timing Sheet (Section 6) |

|  |  |
| --- | --- |
| **STEP 7** | **Competition Final & Awards** |
|  | * Each pilot's best score is ranked on the class leaderboard * Awards: fastest overall time; fewest penalties; best landing (closest to zone centre) * Class debrief: which element caused the most penalties across all pilots? Design a targeted practice drill for it |

**5. Engineering Principles**

**Course Elements & Skills**

| **Element** | **Description** | **Dimensions** | **Skill Tested** |
| --- | --- | --- | --- |
| **Gate A** | Vertical frame; fly through horizontally | 1.5 m wide × 1 m tall | Lateral precision |
| **Gate B** | Vertical frame; fly through horizontally | 1.2 m wide × 1 m tall | Precision at speed |
| **Gate C** | Elevated gate | 1.2 m wide × 1.2 m tall | Altitude management |
| **Slalom 1–4** | Four pylons; weave between them | 1 m spacing | Direction change rate |
| **Landing Zone** | Marked circle on ground | 1 m diameter | Final precision & control |

**Penalty System**

| **Infraction** | **Time Penalty** | **Notes** |
| --- | --- | --- |
| **Touch a gate frame** | + 5 seconds | Confirmed by spotter on that gate |
| **Miss a gate (fly around it)** | + 10 seconds | Gate must be re-flown; add 10 s for missed attempt |
| **Touch the ground outside landing zone** | + 3 seconds | Per touch |
| **Land outside the landing zone** | + 5 seconds | Pilot must take off and re-land within the zone |
| **Hard landing (tip-over)** | + 5 seconds + repair time | Stop clock while repairs made |
| **Fly outside course boundary** | Run disqualified | Must restart from Gate A |

|  |
| --- |
| **Look-Ahead Technique – The Key to Fast, Clean Flying**  Skilled pilots do not look at what they are flying through — they look at what comes NEXT.  By the time the drone reaches Gate A, the pilot's eyes should already be on Gate B.  This gives the pilot time to set up the approach angle, altitude, and speed for the next element.  Looking at the current gate causes late corrections, overshoots, and missed turns.  This technique is identical to the advanced scan technique taught to commercial pilots: 'never fixate; always scan ahead'. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Consistently hits gates** | Flying too fast for skill level; poor approach line | Slow down significantly; practise each gate individually first |
| **Misses turns between gates** | Not planning the route ahead | Look at the next gate before reaching the current one |
| **Loses altitude in slalom** | Over-concentrating on lateral movement; forgetting throttle | Set a base throttle before entering slalom; make only lateral corrections |
| **Hard landing** | Descending too fast on landing approach | Reduce throttle earlier; approach at a shallow angle; flare before touchdown |
| **Over-controls after gates** | Correction inputs too large | Make smaller inputs; accept slight imprecision; smooth > precise |
| **Crashes on first run** | Attempting full speed before skills are ready | Walk the course; fly each element slowly twice before timed run |

**9. Upgrade & Extension Ideas**

* Multi-Lap Course – Fly the course twice in succession without landing; total time for both laps
* Team Relay – Two pilots; pilot 1 flies Gates A and B; hands to pilot 2 for Gate C, slalom, and landing
* Night Course – Fit LEDs to the gate frames; fly the course under low-light conditions
* FPV Course – All timed runs flown using FPV goggles (Project 15 prerequisite)
* Payload Race – Carry the Project 17 cargo hook payload through the course; release it in the landing zone

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* Build the gates before Lesson 1 — PVC pipe gates take 20 minutes to assemble and should be done by the teacher in advance
* Tape the gate frames with bright tape so gate spotters can clearly see when the drone touches
* For the first timed run, have the teacher narrate: 'Looking good through A; set up for B now...' — this models the look-ahead technique
* After the competition, display the class leaderboard publicly — it motivates improvement and creates a positive competitive culture

**Differentiation Strategies**

* Support – 2 gates + slalom only; larger gates (2 m wide); 120-second target; no penalty system
* Core – Full 3-gate + slalom course; penalty system; 3 timed runs; 90-second target
* Extension – Multi-lap; FPV course; payload race; team relay
