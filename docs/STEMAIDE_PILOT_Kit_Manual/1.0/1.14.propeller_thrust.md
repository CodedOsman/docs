**Student & Teacher Manual**

**1. Project Overview**

Propeller thrust is the fundamental output of any propulsion system — and measuring it quantitatively separates engineering from guesswork. Students build a thrust measurement rig using a DC motor, three propellers of different sizes, and a spring scale, then systematically test each propeller across three motor speeds. By calculating efficiency (thrust per ampere of current), students discover that bigger is not always better — and why aircraft engineers carefully match propeller size to motor and mission.

**Learning Objectives**

* Build a stable thrust measurement rig with a DC motor, spring scale, and three test propellers
* Explain the relationship between propeller diameter, pitch, and thrust output
* Test three propeller sizes at three motor speeds; record thrust and current for each combination
* Calculate propeller efficiency as thrust (g) per ampere of current drawn
* Plot a thrust vs. motor speed graph for all three propellers on a single graph
* Identify the most efficient propeller for a given mission profile

**3. Build Steps & Assembly**

**Lesson 1 – Rig Construction**

|  |  |
| --- | --- |
| **STEP 1** | **Base Construction** |
|  | * Mount the motor on a wooden block or dowel at one end of the base board * The motor shaft must point horizontally away from the base board (forward thrust direction) * Secure the motor mount firmly — any movement will give false thrust readings * Clamp the base board to the table with a G-clamp before any motor testing |

|  |  |
| --- | --- |
| **STEP 2** | **Thrust Measurement Setup** |
|  | * Mount the spring scale vertically at the opposite end of the base board * Connect motor to spring scale with a 15 cm length of string; string must be horizontal and taut * When the motor generates thrust, the string will pull the spring scale — the reading is the thrust * Zero the spring scale with the string connected but motor off |

|  |  |
| --- | --- |
| **STEP 3** | **Electrical Connection** |
|  | * Connect: battery holder → inline switch → inline ammeter → motor → back to battery * Confirm polarity: motor should spin to push air away from the base board (forward thrust) * Test with the smallest propeller at low speed: spring scale should show a positive reading * If scale reads zero or negative: reverse the two motor wires |

**Lessons 2–3 – Propeller Testing**

|  |  |
| --- | --- |
| **STEP 4** | **Test Protocol** |
|  | * REMOVE ALL PROPELLERS before making any electrical connections — only fit a propeller immediately before a test * Set motor to Low speed (mark the potentiometer or use minimum battery voltage) * Fit Small propeller; switch on; wait 3 seconds for speed to stabilise; read spring scale; read ammeter * Record both values; switch off; wait 10 seconds before changing propeller * Repeat for Medium and Large propellers at the same speed setting * Repeat the full sequence at Medium and High speed settings |

**Lesson 4 – Analysis & Graphing**

|  |  |
| --- | --- |
| **STEP 5** | **Calculate Efficiency & Plot** |
|  | * Efficiency = Thrust (g) ÷ Current (A) — higher is better * Plot a graph: Motor Speed (x-axis: Low/Medium/High) vs. Thrust in grams (y-axis) * Plot all 3 propellers as separate lines on the same graph * Identify: which propeller is most efficient at low speed? At high speed? * Write a conclusion: for a long-endurance glider (slow, efficient), which propeller would you choose? |

**4. Power & Safety Notes**

|  |

| **⚠ Safety Requirements**  Power: 3V (2× AA batteries) — safe for students to handle.  Propellers: Safety goggles MANDATORY during all motor testing. A propeller shard at full speed is a serious eye hazard.  Propeller guard: Attach a wire cage around the propeller zone — propellers must never be at face height without a guard.  Rig stability: Clamp the base board before EVERY test run — thrust can cause the rig to move.  NEVER touch a spinning propeller. Wait until it has completely stopped before changing propellers. |

**5. Engineering Principles**

* Thrust: force generated by the propeller accelerating a mass of air rearward; Newton's Third Law — equal and opposite reaction pushes the aircraft forward
* Diameter: larger diameter propellers move more air per revolution, generating more thrust at the same RPM but requiring more torque from the motor
* Pitch: the angle of the propeller blade; high pitch moves more air per revolution but needs more power; low pitch needs less power but produces less thrust
* Efficiency: the ratio of useful thrust to electrical power consumed. A large, slow-spinning propeller is more efficient than a small, fast-spinning one — why all long-endurance aircraft use large propellers

| **Real Aviation Connection**  This rig is a simplified version of the thrust test benches used by propeller manufacturers.  Every propeller fitted to a certified aircraft must pass a documented thrust and efficiency test.  The most efficient propellers on earth are on long-endurance UAVs like the AeroVironment Global Observer — 2-metre span blades turning at 300 RPM, designed using exactly the principles measured in this project. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Motor does not spin** | Wrong wiring or open circuit | Check all connections; confirm battery polarity; test with a multimeter |
| **Rig moves during testing** | Not clamped to table | Clamp the base board to the table with a G-clamp; retest |
| **Inconsistent thrust readings** | Propeller unbalanced | Balance propeller using a prop balancer; sand heavier blade lightly |
| **Low thrust on all propellers** | Motor direction reversed | Swap any two motor wires; confirm propeller pushes air rearward |
| **Spring scale bounces** | Vibration from unbalanced propeller | Increase balancing; read scale at peak of oscillation average |

**9. Upgrade & Extension Ideas**

* Variable Speed Control: add a potentiometer (variable resistor) to set precise speed levels; measure RPM with a phone tachometer app
* Voltage Test: run the same propeller at 3V, 4.5V, and 6V; graph voltage vs. thrust
* Counter-Rotating Twin: mount two motors facing opposite directions with identical propellers; measure combined thrust vs. single motor
* Ducted Fan: wrap a cardboard duct around the propeller; measure the thrust increase from the duct effect

**10. Teacher Notes & Differentiation**

* Clamp the rig to the table before every single test — this is non-negotiable and prevents both injury and false readings
* The ammeter connection (Step 3) is the most technically challenging part — have a pre-wired demonstration circuit ready
* Support – Pre-wired rig; two propellers only; no current measurement; focus on reading thrust values and plotting one graph
* Core – Full build; all 3 propellers; 3 speed settings; efficiency calculation; comparative graph
* Extension – Variable speed control; voltage test; ducted fan
