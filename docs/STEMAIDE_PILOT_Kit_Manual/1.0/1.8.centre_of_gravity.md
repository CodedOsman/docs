**Student & Teacher Manual**

**1. Project Overview**

Every aircraft has a centre of gravity — the single point at which its entire weight appears to act. If that point is in the wrong place, the aircraft will be unstable or uncontrollable. Students use a precision balance beam and sliding weights to find the CG of three foam aircraft models, classify each model's stability, and investigate how payload loading shifts the CG. This is the same analysis performed by load-and-trim engineers before every commercial flight departure.

**Learning Objectives**

* Define centre of gravity (CG) and explain its relationship to the aerodynamic centre (AC)
* Use a balance beam and sliding weights to accurately locate the CG of three foam models
* Classify each model as statically stable, neutral, or unstable based on observed behaviour
* Investigate how adding payload at different positions shifts the CG
* Explain why commercial aircraft use load-and-trim sheets before every departure

**3. Lesson Guide & Steps**

**Lesson 1 – Balance Beam Setup & CG Finding**

|  |  |
| --- | --- |
| **STEP 1** | **Balance Beam Assembly & Levelling** |
|  | * Place balance beam on a flat, stable desk * Set pivot at the marked centre point * Check beam is level using a set square — beam should balance horizontally with no models loaded * If beam tips, move pivot slightly toward the heavier end until level * Lock pivot once balanced |

|  |  |
| --- | --- |
| **STEP 2** | **Sliding Weight Calibration** |
|  | * Slide all three weights to the exact centre mark * Confirm beam still balances — recheck pivot if not * Practice: slide one weight toward one end; observe the beam tip * Label the beam in 1 cm intervals from pivot using ruler and marker * Return all weights to centre before proceeding |

|  |  |
| --- | --- |
| **STEP 3** | **Verify CG Mark on Each Model** |
|  | * Each foam model has a CG dot marked on its underside * Verify by balancing each model on one fingertip under the wing * Fingertip balance point should match the marked dot * Record CG as distance from nose of each model |

|  |  |
| --- | --- |
| **STEP 4** | **Find CG Using the Balance Beam** |
|  | * Place Model A on beam with nose at the zero mark * Slide weights away from centre until beam balances level * Record weight positions; the balance point is the CG * Mark position with tape labelled 'Model A CG' * Repeat for Models B and C |

**Lesson 2 – Stability Testing & Load Experiment**

|  |  |
| --- | --- |
| **STEP 5** | **Stability Classification Test** |
|  | * Balance each model on beam at its confirmed CG point * Gently tilt model's nose down by 5° — apply only light pressure * Release completely and observe response: * Returns to level → STABLE (CG forward of aerodynamic centre) * Stays at 5° → NEUTRAL (CG at aerodynamic centre) * Continues to tip → UNSTABLE (CG behind aerodynamic centre) * Record stability classification for each model |

|  |  |
| --- | --- |
| **STEP 6** | **Load Distribution Experiment** |
|  | * Take Model A; tape one coin to the nose area * Re-find CG using balance beam; record new position * Test stability; record result * Move coin to the tail; repeat CG finding and stability test * Move coin to wing centre; repeat * Complete the Load Experiment Table for all four scenarios |

|  |  |
| --- | --- |
| **STEP 7** | **Results Analysis & Discussion** |
|  | * Compare CG positions across all three models — which has the most forward CG? * Discuss: Why are fighter jets designed to be slightly unstable? What technology compensates? * Students write a 3-sentence summary: What is CG? Why does position matter? What did we find? * Teacher connects to real aviation: aircraft CG limits are printed in the flight manual; exceeding them is illegal |

**5. Engineering Principles**

**Centre of Gravity vs. Aerodynamic Centre**

The Centre of Gravity (CG) is where all of the aircraft's weight acts downward. The Aerodynamic Centre (AC) is where all the lift force effectively acts. Their relative positions determine whether the aircraft is stable, neutral, or unstable.

|  |
| --- |
| **The Golden Rule of Aircraft Stability**  For static stability: CG must be AHEAD of (forward of) the Aerodynamic Centre.  If the nose drops, a forward CG creates a restoring moment — the aircraft pitches back to level.  If the CG is behind the AC, a nose drop causes the aircraft to pitch further down — unrecoverable dive. |

**Stability Classification**

| **Stability Type** | **Behaviour After Disturbance** | **Aircraft Example** | **CG Position** |
| --- | --- | --- | --- |
| **Stable** | Returns to original level position | Trainer aircraft; commercial jets | Forward of aerodynamic centre |
| **Neutral** | Stays at new disturbed position | Some aerobatic aircraft | At aerodynamic centre |
| **Unstable** | Diverges further from original | Fighter jets (fly-by-wire corrects) | Behind aerodynamic centre |

**Real Aviation – Weight & Balance**

* Every commercial flight has a Load and Trim Sheet prepared by engineers before departure
* The sheet calculates CG from passenger seat assignments, cargo loading, and fuel quantity
* If calculated CG falls outside certified limits, cargo must be moved before departure
* Ghana's Africa World Airlines, like all ICAO carriers, must file a Weight & Balance manifest for every flight

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Beam will not balance empty** | Pivot not centred or surface uneven | Re-centre pivot; place on a flat, level surface |
| **Aircraft keeps tipping off beam** | CG not at balance point | Slide weights slowly; pause and check level frequently |
| **Stability misclassified** | Push applied with too much force | Apply only a gentle 5° tilt; release fully and observe response |
| **Inconsistent results** | Model repositioned differently each time | Mark nose position on beam with tape for consistency |
| **Weights slide during stability test** | Locking screw not tightened | Tighten wing-nut on each weight after positioning |

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* Pre-set the beam to Model A's known CG before Lesson 1 — students see a correctly balanced example immediately
* The stability test (Step 5) requires a very gentle push — demonstrate the correct force before students attempt
* Use coins as payload: 1 GHS coin ≈ 4 g — consistent and easy to count
* Show a photo of a real commercial load-and-trim sheet — students are often surprised by the detail

**Differentiation Strategies**

* Support – Focus on Models A and B only; teacher assists with beam; binary classification (stable/unstable only)
* Core – All 3 models; load experiment with 4 scenarios; 3-sentence summary
* Extension – CG calculator spreadsheet; Weight & Balance form design; real aircraft CG limit research

**Assessment Suggestions**

* Data table: CG positions to within 0.5 cm; stability correctly classified
* Load experiment: All 4 scenarios completed; CG shift correctly noted
* Summary: CG-stability relationship explained correctly in own words
* Discussion: Student can explain why fighter jets are intentionally unstable
