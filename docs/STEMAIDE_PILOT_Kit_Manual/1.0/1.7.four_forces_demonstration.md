**Student & Teacher Manual**

**1. Project Overview**

This project makes the invisible forces of flight visible. Using a desk fan to generate controlled airflow and lightweight streamers to trace flow patterns, students directly observe how different wing shapes interact with moving air. By building a four-forces wall chart and participating in a class demonstration, students move from abstract theory to concrete, observable aerodynamics.

|  |  |
| --- | --- |
| Curriculum Area | Aviation Physics – Aerodynamics & Forces of Flight |
| Year Group | SHS 1 (Terms 1–2) |
| Duration | 3 lessons of 40–50 minutes each |
| Materials Source | Kit 1 (poster, markers) and Kit 3 (fan, airfoils, streamers) |
| Power Required | 220 V desk fan – teacher-operated only |
| Prerequisite | Projects 1–2 (basic aerodynamics familiarity) |

**Learning Objectives**

* Identify and explain all four forces of flight: lift, weight, thrust, and drag
* Observe and describe the difference between laminar and turbulent airflow using streamers
* Compare how symmetric, cambered, and flat-plate airfoils perform at the same airspeed
* Apply Bernoulli's Principle to explain why a cambered wing generates lift at zero angle of attack
* Create an annotated four-forces wall chart and airflow diagram
* Present airfoil comparison findings to the class

**3. Setup & Lesson Steps**

**Before Lesson 1 – Fan & Streamer Station Setup**

|  |  |
| --- | --- |
| **STEP 1** | **Fan & Streamer Station** |
|  | * Place the desk fan on a stable table at bench height — do NOT switch on yet * Stretch masking tape horizontally across a frame 30 cm in front of the fan outlet * Attach 8–10 streamers at 5 cm intervals along the tape so they hang freely * Label three floor zones: Low Speed, Medium Speed, High Speed for different fan settings * Set out the three airfoil models, clearly labelled |

**Lesson 1 – Airflow Observation**

|  |  |
| --- | --- |
| **STEP 2** | **Streamer Baseline Observation** |
|  | * Teacher switches fan to low speed; class observes streamers with no airfoil present * Students sketch the streamer pattern: straight parallel lines = laminar flow * Teacher increases to medium speed; note how streamers respond * Introduce vocabulary: laminar flow (smooth, parallel) vs. turbulent flow (chaotic, separated) |

|  |  |
| --- | --- |
| **STEP 3** | **Airfoil Testing – One at a Time** |
|  | * Place Symmetric airfoil in the airstream at zero angle of attack (level) * Students observe and sketch streamer pattern above and below the airfoil * Are streamers above moving faster or slower than those below? * Repeat with Cambered airfoil; compare top vs. bottom streamer speed * Repeat with Flat Plate; note where turbulence appears (behind trailing edge) * Record all observations in the Data Collection Table (Section 6) |

|  |  |
| --- | --- |
| **STEP 4** | **Angle of Attack Experiment** |
|  | * Take Symmetric airfoil; tilt upward 10° — observe streamers above speed up * Tilt to 20°; observe streamers behind wing become turbulent — stall beginning * Record: at what angle did turbulence first appear? * Repeat with Cambered airfoil; note it produces lift even at 0° |

**Lesson 2 – Four Forces Wall Chart**

|  |  |
| --- | --- |
| **STEP 5** | **Build the Four Forces Wall Chart** |
|  | * Draw a large side-view aircraft silhouette in the centre of the A1 poster (25+ cm wingspan) * Draw four arrows from the aircraft centre, each a different colour: * Red arrow UP → label: LIFT * Blue arrow DOWN → label: WEIGHT * Green arrow FORWARD → label: THRUST * Orange arrow BACKWARD → label: DRAG * Arrow length reflects relative magnitude — Lift ≈ Weight for level flight * Add at least 2 real-world facts per force |

|  |  |
| --- | --- |
| **STEP 6** | **Annotate with Bernoulli's Principle** |
|  | * On a separate section, draw a cross-section of the cambered airfoil * Show flow lines: closer together (faster) above; wider apart (slower) below * Label: 'Lower Pressure' above; 'Higher Pressure' below * Add a pressure difference arrow pointing upward — label: LIFT * Write Bernoulli's Principle in own words below the diagram |

**Lesson 3 – Demonstration & Presentation**

|  |  |
| --- | --- |
| **STEP 7** | **Class Demonstration** |
|  | * Each group presents their wall chart (3–4 minutes per group) * One student operates fan; another holds airfoil; third points to streamers * Presenting student explains: (1) which airfoil produces most lift and why, (2) what happens at high angles of attack * Class asks one question per group |

|  |  |
| --- | --- |
| **STEP 8** | **Angle of Attack Test – Quantitative Extension** |
|  | * Mount cambered airfoil on a protractor stand — set angle precisely * Test at 0°, 5°, 10°, 15°, 20°; record streamer behaviour at each angle * Create a bar graph: Angle of Attack vs. Flow Quality (1=laminar → 5=separated) * Identify the critical (stall) angle where flow completely separates |

**5. Engineering Principles**

**The Four Forces of Flight**

| **Force** | **Direction** | **Source** | **Real Aircraft Example** |
| --- | --- | --- | --- |
| **Lift** | Upward | Curved wing (Bernoulli + angle of attack) | Wing generates lift in forward flight |
| **Weight** | Downward | Gravity acting on aircraft mass | Heavier aircraft needs more lift |
| **Thrust** | Forward | Engine, propeller, or hand-launch | Fan simulates airstream; hand = thrust |
| **Drag** | Backward | Air resistance on all surfaces | Streamlined shapes reduce drag |

**Airfoil Comparison**

| **Airfoil** | **Streamer Pattern** | **Lift at 0°** | **Use Case** |
| --- | --- | --- | --- |
| **Symmetric** | Even flow top & bottom | Minimal | Aerobatic aircraft; helicopter rotor |
| **Cambered** | Faster top, slower bottom | Good | Commercial aircraft; gliders; trainers |
| **Flat Plate** | Turbulent behind trailing edge | Poor | Educational baseline; kites |

|  |
| --- |
| **Bernoulli's Principle**  When air speeds up, its pressure decreases.  A cambered wing forces air over a longer curved path on top — air speeds up — pressure drops.  Higher pressure below the wing pushes upward: this is LIFT.  At high angles of attack, airflow separates from the surface — this is a STALL. |

**Why Three Airfoil Types?**

* Symmetric: Used on aerobatic aircraft and helicopter rotors — no lift bias; flies equally inverted
* Cambered: Used on most trainers, gliders, and commercial aircraft — efficient lift at cruise speeds
* Flat plate: Educational baseline showing the consequence of no airfoil shaping — high drag, poor lift

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Streamers barely move** | Fan too far away or speed too low | Move airfoil closer; increase fan speed |
| **No visible lift difference between airfoils** | Airfoil not oriented correctly | Ensure cambered airfoil curved side faces upward into airstream |
| **Force diagram arrows reversed** | Confusion over direction conventions | Review: Lift=up, Weight=down, Thrust=forward, Drag=backward |
| **Turbulence at 0° on cambered foil** | Airfoil inverted | Flip so curved surface faces the fan outlet |

**9. Upgrade & Extension Ideas**

* Wind Tunnel – Build a simple enclosed tunnel with clear plastic sides; compare streamer patterns
* Smoke Visualisation – Use incense smoke in a ventilated area for sharper flow lines
* Spring Scale Lift – Mount cambered airfoil on a spring scale; measure actual upward force at different fan speeds
* Aspect Ratio Experiment – Cut two wings of equal area but different span/chord ratios; compare lift and drag
* Digital Video Annotation – Record streamer footage; annotate using a free video editing app

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* Secure the fan to the table with a G-clamp — fans vibrate and can fall
* Pre-label all three airfoil models clearly before the lesson
* Lesson 3 presentations work best with groups of 4: fan operator, airfoil holder, streamer pointer, presenter
* Photograph each group's wall chart for the school aviation display board

**Differentiation Strategies**

* Support – Pre-drawn force diagram template; two airfoil types only; teacher-led streamer observation
* Core – Full three-airfoil comparison; force wall chart; class presentation
* Extension – Quantitative angle-of-attack test with graph; spring scale lift measurement

**Assessment Suggestions**

* Data table: All 3 airfoils tested; observations in own words
* Wall chart: All 4 forces correctly labelled with proper arrow directions
* Bernoulli annotation: Correct explanation linking faster airflow → lower pressure → lift
* Presentation: Student explains airfoil comparison results correctly
