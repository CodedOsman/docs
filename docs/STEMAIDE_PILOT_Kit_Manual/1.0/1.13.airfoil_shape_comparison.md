**Student & Teacher Manual**

**1. Project Overview**

This project transforms the qualitative airfoil observations from Project 7 into quantitative measurements. Using a spring scale attached to a foam airfoil test rig in front of the Kit 3 desk fan, students measure actual lift forces in grams for four different airfoil shapes across four angles of attack. The resulting data allows students to plot lift curves — the same curves aeronautical engineers produce during wind tunnel testing — and identify the stall angle for each design.

**Learning Objectives**

* Build four foam airfoil models: symmetric, cambered, flat plate, and under-cambered
* Assemble a test rig with a pivoting airfoil mount and spring scale to measure lift force in grams
* Test each airfoil at 0°, 5°, 10°, and 15° angles of attack and record lift forces
* Plot a lift curve (lift vs. angle of attack) for all four airfoils on a single graph
* Identify the stall angle for each airfoil and explain the mechanism of stall
* Select the best-performing airfoil for a specific mission and justify using the data

**3. Build Steps & Assembly**

**Lesson 1 – Airfoil Construction**

|  |  |
| --- | --- |
| **STEP 1** | **Build Four Foam Airfoils** |
|  | * Trace the four airfoil templates onto foam using a marker * Symmetric: equal curvature top and bottom; cut from one folded foam sheet * Cambered: curved top surface, flat bottom; use the score-and-fold technique from Project 11 * Flat Plate: simple rectangle with square leading and trailing edges * Under-Cambered: flat top, curved bottom (opposite of cambered) * Sand all leading edges to a rounded profile; sand trailing edges to a thin taper * Each airfoil must be 15 cm span × 5 cm chord; label on the underside |

|  |  |
| --- | --- |
| **STEP 2** | **Build the Test Rig** |
|  | * Mount a 20 cm wooden dowel vertically on the test base (cardboard box or wooden board) * Drill or poke a horizontal hole through the dowel at 10 cm height — this is the pivot * Insert a 15 cm horizontal rod through the pivot so it can rotate freely * Attach the airfoil to the horizontal rod so it is centred on the pivot * Attach the spring scale with string from the trailing edge of the airfoil (measures lift as upward pull) * Use the protractor to mark 0°, 5°, 10°, 15°, 20° angles on the pivot mount * Position the fan 20 cm from the airfoil leading edge; ensure it is at the same height |

**Lesson 2 – Data Collection**

|  |  |
| --- | --- |
| **STEP 3** | **Test Protocol** |
|  | * Zero the spring scale with airfoil mounted at 0° before turning on the fan * Turn fan to a fixed speed; allow 5 seconds to stabilise before reading the scale * Record the lift reading; rotate airfoil to 5°; wait 5 s; record; continue to 10°, 15°, 20° * Repeat all angles for all four airfoils — keep fan speed constant throughout * Record all values in the data table immediately; do not rely on memory |

**Lessons 3–4 – Analysis & Graphing**

|  |  |
| --- | --- |
| **STEP 4** | **Plot Lift Curves** |
|  | * On graph paper: x-axis = angle of attack (0° to 20°); y-axis = lift in grams * Plot all four airfoils on the same graph; use a different colour for each * Draw smooth curves through the data points (not just dot-to-dot) * Identify the peak lift point for each airfoil — this is the maximum lift angle * Mark where lift drops sharply — this is the stall angle |

|  |  |
| --- | --- |
| **STEP 5** | **Analysis & Conclusion** |
|  | * Which airfoil produces the most lift at 0° angle of attack? (should be cambered) * Which airfoil has the highest stall angle? (usually symmetric) * Which would you choose for: (a) a glider? (b) an aerobatic aircraft? (c) a slow cargo plane? * Write a 3-paragraph conclusion: best airfoil, why it is best, what the data shows |

**5. Engineering Principles**

* Bernoulli's Principle: a cambered airfoil forces air to travel farther over its curved top surface, so it must travel faster; faster flow = lower pressure; lower pressure above = lift
* Angle of attack: the angle between the chord line and the oncoming airflow; increasing AoA increases lift up to the stall angle
* Stall: above the critical angle of attack, airflow cannot follow the upper surface; it separates; pressure equalises; lift drops suddenly
* Under-cambered airfoils: popular on slow-flying model aircraft and kites — high lift at low speeds; poor at higher speeds due to drag

|  |
| --- |
| **Real Wind Tunnel Connection**  Every aircraft airfoil in the world has been tested in a wind tunnel before being certified for flight.  The data students collect in this project is structurally identical to wind tunnel data: lift coefficient vs. angle of attack.  The Airbus A380 wing profile was tested at subsonic, transonic, and supersonic speeds before the aircraft was built.  Students are conducting the same class of experiment using a desk fan instead of a million-dollar facility. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **No lift measured** | Fan too far from airfoil | Move fan to within 20 cm of the airfoil leading edge |
| **Inconsistent readings** | Fan speed varies between tests | Mark the fan speed dial and return to same position each test; allow 5 s to stabilise |
| **Airfoil twists under airload** | Mount too flexible | Reinforce the mounting rod; use a stiffer cardboard backing |
| **Scale reads negative (downforce)** | Airfoil mounted inverted | Check cambered surface is facing upward into the airstream |
| **No stall observed** | Angle not increased enough | Continue increasing to 20°, 25°; stall is usually between 15° and 20° |

**9. Upgrade & Extension Ideas**

* Wind Tunnel Enclosure: build a cardboard enclosure around the test area to produce a more uniform airstream
* Flap Effect: score and bend the trailing 20% of the cambered airfoil down as a flap; measure lift change
* Aspect Ratio: cut two wings of equal area — one long and narrow, one short and wide; compare lift at the same angle
* Electronic Measurement: replace the spring scale with a force sensor connected to an ESP32; log data automatically

**10. Teacher Notes & Differentiation**

* The test rig is the critical build — spend time in Lesson 1 confirming it pivots freely and the scale reads correctly before foam airfoils are made
* Fan speed consistency is essential — if the fan has no fixed-speed dial, power it through a variable resistor or simply ensure students turn it to the same marked position each time
* Support – Pre-built airfoils; 2 designs only (cambered and flat plate); 0° and 10° only; no graph
* Core – All 4 airfoils; all angles; graph; stall identification; written conclusion
* Extension – Flap effect; aspect ratio; electronic data logging
