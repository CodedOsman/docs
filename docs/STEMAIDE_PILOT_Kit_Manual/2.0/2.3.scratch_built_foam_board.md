**Student & Teacher Manual**

**1. Project Overview**

This is the flagship project of the Intermediate section. Students design and build a complete, flyable radio-controlled aircraft from scratch using foam board — the material of choice for scratch-build designers worldwide. From template tracing and airfoil creation through servo installation, ESC wiring, and power system integration to maiden flight, this project synthesises every skill learned in Projects 1–10 into a single flying aircraft.

|  |  |
| --- | --- |
| Curriculum Area | RC Aircraft Design, Construction & Flight Operations |
| Year Group | SHS 1 Term 3 – SHS 2 Term 2 (Intermediate) |
| Duration | 8–10 lessons of 40–50 minutes each |
| Materials Source | Kit 1 (foam, tools), Kit 5 (motor), Kit 6 (servos, pushrods), Kit 10 (ESC, receiver, battery, transmitter) |
| Power Required | LiPo 2S or 3S battery – teacher-managed |
| Prerequisite | Projects 1–10 completed; strong CG and control surface skills from Projects 2, 5, and 8 |

**Learning Objectives**

* Design and cut a complete foam board airframe to a provided plan
* Create a functional foam board airfoil using the score-and-fold technique
* Install three servo motors and connect them to control horns via pushrods
* Wire the brushless motor, ESC, receiver, and battery into a complete power system
* Balance the aircraft to the correct CG and verify all control directions
* Complete an unpowered glide test followed by a teacher-supervised maiden flight

**3. Build Steps & Assembly**

**Design Parameters**

| **Parameter** | **Recommended Value** | **Reason** |
| --- | --- | --- |
| **Wing span** | 60–70 cm | Enough lift for 2S battery and motor weight |
| **Chord (wing width)** | 15–18 cm | Aspect ratio of ~4:1 — efficient for slow flight |
| **Dihedral angle** | 5–10° | Provides passive roll stability |
| **Control surface size** | 20% of wing / tail area | Enough authority without over-sensitivity |
| **CG position** | 30–33% chord from leading edge | Forward of aerodynamic centre for stability |

**Lessons 1–2 – Airframe Design & Cutting**

|  |  |
| --- | --- |
| **STEP 1** | **Template Tracing** |
|  | * Print or hand-draw the plan at 1:1 scale onto foam board using pencil * Parts to trace: Main wing, fuselage sides, fuselage top & bottom, horizontal stabilizer, vertical stabilizer, ailerons (×2), elevator (×1) * Mark fold lines with a dashed pencil line — do not cut these * Label each piece before cutting |

|  |  |
| --- | --- |
| **STEP 2** | **Cutting & Wing Airfoil** |
|  | * Cut all parts with craft knife and steel ruler on cutting mat * For wing airfoil: score (do not cut through) upper surface along marked fold line * Fold foam over score line to create curved upper surface — this is the airfoil * Apply hot glue along the inner fold to hold the shape * Sand leading edge to rounded profile; taper trailing edge to a sharp line |

|  |  |
| --- | --- |
| **STEP 3** | **Fuselage Assembly** |
|  | * Glue two fuselage side panels together at nose and tail * Insert and glue top and bottom panels to form a box structure * Cut motor mount hole in nose panel — propeller must clear fuselage by 5 mm minimum * Reinforce nose motor mount area with packing tape on both sides * Cut battery bay opening — battery must slide in and out easily |

**Lessons 3–4 – Control System Installation**

|  |  |
| --- | --- |
| **STEP 4** | **Servo Installation** |
|  | * Cut servo pockets in wing (ailerons) and tail (elevator) using servo as template * Insert each servo so output arm is level with control surface hinge line * Secure with hot glue on three sides — leave one side accessible for re-gluing * Route servo wires along wing spar and fuselage spine toward receiver bay |

|  |  |
| --- | --- |
| **STEP 5** | **Control Linkage Assembly** |
|  | * Bend a Z-bend at one end of each pushrod using pliers * Insert Z-bend into outermost hole of control horn * Fit other end through servo output arm's outer hole using clevis or Z-bend * Adjust length until control surface is exactly neutral with servo centred * Test: servo moved by hand; surface moves freely with no binding |

|  |  |
| --- | --- |
| **STEP 6** | **Power System Integration** |
|  | * Mount brushless motor to motor mount; torque M3 screws firmly * Connect ESC output leads to motor (three wires — direction confirmed later) * Mount receiver centrally with double-sided foam tape * Connect servo leads: aileron servos → channels 1 & 2; elevator → channel 3; ESC throttle → channel 4 * Route ESC power lead to battery connector — do NOT connect battery yet |

**Lessons 5–6 – Final Assembly, CG & Checks**

|  |  |
| --- | --- |
| **STEP 7** | **Wing Attachment & Dihedral** |
|  | * Join wing halves with a 5° upward angle at centre (dihedral) * Reinforce wing join with packing tape on upper and lower surfaces * Attach wing perpendicular to fuselage — no sweep or droop * Attach horizontal and vertical stabilizers; verify they are square to each other and to the wing |

|  |  |
| --- | --- |
| **STEP 8** | **CG Check & Ballast** |
|  | * Balance aircraft on fingertips at 30–33% chord from leading edge (with battery in flight position) * Nose-heavy: move battery back or add tail weight * Tail-heavy: move battery forward or add nose weight * The CG with battery installed is the real flight CG — confirm this position |

|  |  |
| --- | --- |
| **STEP 9** | **Control Direction Verification** |
|  | * Bind transmitter; teacher connects battery (motor disconnected for safety) * Verify all control directions using the Control Check Table (Section 6) * Correct reversed channels via transmitter servo-reverse menu * Set control throws: 15 mm deflection each way for first flight |

|  |  |
| --- | --- |
| **STEP 10** | **Glide Test (No Power)** |
|  | * Remove battery; weight nose to normal all-up weight with coin if needed * Launch at gentle downward angle; observe flight path * Must fly straight for 5+ metres without turning or diving * Adjust CG or rudder trim until straight glide confirmed * Only proceed to powered flight after glide test passes |

|  |  |
| --- | --- |
| **STEP 11** | **Maiden Flight** |
|  | * Teacher connects battery and arms ESC * Pilot launches with a smooth, level forward throw — do not push upward * Apply 60% throttle immediately after release * Fly a gentle left-hand circuit; make small control inputs * Land with slow throttle-off glide; flare gently just before touchdown |

**4. Power & Safety Notes**

|  |
| --- |
| **⚠ Safety Requirements**  Power: LiPo 2S or 3S battery. Teacher handles all battery connections.  Propeller: Keep clear of the propeller arc at all times while battery is connected.  Eye protection: Safety goggles mandatory during motor testing and all flights.  Flight zone: Designated outdoor area; minimum 20 m × 40 m; clear of all bystanders.  Control check MUST be completed before any powered test — a reversed elevator causes an immediate crash. |

**5. Engineering Principles**

**Control Surfaces & Inputs**

| **Surface** | **Servo Location** | **Stick Input** | **Aircraft Response** |
| --- | --- | --- | --- |
| **Ailerons** | Outer wing trailing edge | Right stick Left/Right | Roll left or right |
| **Elevator** | Horizontal stabilizer trailing edge | Right stick Up/Down | Pitch nose up or down |
| **Rudder (optional)** | Vertical stabilizer trailing edge | Left stick Left/Right | Yaw nose left or right |
| **Throttle** | ESC connected to motor | Left stick Up/Down | Increase or decrease speed |

|  |
| --- |
| **Score-and-Fold Airfoil**  Foam board has a paper skin on both sides and a foam core.  Scoring — cutting through one paper layer without cutting the foam — allows the board to curve.  When scored face is on the inside of the fold, the outer paper skin creates a smooth curved surface.  This produces a simple but effective cambered airfoil that generates real lift. |

**7. Expected Output & Success Criteria**

| **Outcome** | **Success Criteria** |
| --- | --- |
| Airframe built | Foam board cut cleanly; wing and fuselage square; tail aligned |
| Control surfaces installed | All servos respond to transmitter; correct direction verified |
| CG correct | Balances at 30–33% chord; minimal ballast needed |
| Control check complete | All 8 checklist items ticked |
| Glide test passes | Unpowered flight straight and level for 5+ metres |
| Powered flight achieved | Aircraft flies under power for 10+ seconds |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Stalls immediately on launch** | CG too far back or total weight too heavy | Move CG forward; reduce tail weight; check all-up weight |
| **Turns sharply on takeoff** | Wing asymmetry or ailerons not neutral | Check wing symmetry; re-centre ailerons and trim |
| **No response to controls** | Servo disconnected or receiver not bound | Check servo leads; rebind receiver |
| **Motor spins wrong direction** | Two motor wires swapped | Swap any two of the three motor-to-ESC wires |
| **Foam cracks on landing** | Impact too hard or poor foam quality | Reinforce stress joints with packing tape; practise flare landings |

**9. Upgrade & Extension Ideas**

* Flaps – Score and hinge the inner wing trailing edge; connect to a spare servo for landing configuration
* Navigation Lights – Add red (left wing), green (right wing), and white (tail) LEDs wired to a spare channel
* FPV Camera – Add a small camera and video transmitter for first-person-view flight
* Speed Upgrade – Fit a 3S battery and higher-KV motor; redesign wing for faster cruise
* Twin-Motor Design – Build a second motor mount on the other wing tip for a twin-engine configuration

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* Build a completed example aircraft before Lesson 1 — students need to see what they are working toward
* For classes without soldering equipment, use pre-soldered ESC and motor packs with bullet connectors
* The maiden flight should be teacher-operated for the first 30 seconds; hand over once the aircraft is confirmed stable
* Brief all students on hand signals before flight: thumbs up = all clear; flat hand horizontal = freeze

**Differentiation Strategies**

* Support – Pre-cut foam kit; begin from Lesson 3 (servo installation); teacher assists with power system
* Core – Full build from template tracing; all electrical integration; maiden flight
* Extension – Flaps; FPV system; navigation lights; performance data logging
