**Student & Teacher Manual**

**1. Project Overview**

Students assemble a real, flying quadcopter from component level: frame, motors, ESCs, flight controller, receiver, and battery. Every step mirrors the workflow of a professional UAV technician. Safety is the highest priority throughout — propeller guards are mandatory, all first flights are tethered, and batteries are handled exclusively by the teacher until safety protocols are fully established.

|  |
| --- |
| **⚠ IMPORTANT – HIGHEST RISK PROJECT IN THE KIT**  This project involves a real flying vehicle with high-speed rotating propellers and a LiPo battery.  Propeller guards are MANDATORY at all times during testing and flight.  All first flights MUST be tethered to a ground anchor.  The teacher must handle and connect the LiPo battery for all initial tests and flights.  A designated outdoor flight zone, clear of all bystanders, is required for flight activities. |

* Assemble a quadcopter frame with motors, ESCs, flight controller, and receiver
* Explain the role of each component and how they interact to produce controlled flight
* Configure the flight controller using Betaflight software
* Complete a 10-point pre-flight inspection and obtain teacher sign-off before any flight

**3. Build Steps & Assembly**

**Lessons 1–2 – Frame & Motor Assembly**

|  |  |
| --- | --- |
| **STEP 1** | **Frame Assembly** |
|  | * Lay out all frame components: central plate, 4 arms, landing legs, top plate * Attach all four arms to the central plate using M3 screws — finger-tight first, then torque * Check symmetry: measure diagonally arm-tip to arm-tip — both diagonals must be equal * Leave the top plate loose — it will be removed for electronics installation |

|  |  |
| --- | --- |
| **STEP 2** | **Motor Mounting** |
|  | * Identify CW and CCW motors — they are threaded differently * Mount CW motors on Front-Left (M1) and Rear-Right (M3) * Mount CCW motors on Front-Right (M2) and Rear-Left (M4) * Torque all motor screws firmly — loose motors cause vibration * Spin each motor by hand to confirm free rotation with no rubbing |

|  |  |
| --- | --- |
| **STEP 3** | **Propeller Guard Installation** |
|  | * Fit propeller guards around each motor BEFORE any propeller is attached * Guards must be flush with the motor bell; secure with zip ties if needed * Confirm guards do not obstruct the motor shaft * Guards must remain fitted for ALL indoor tests and ALL first-flight outdoor sessions |

**Lessons 3–4 – Electronics Installation**

|  |  |
| --- | --- |
| **STEP 4** | **ESC Wiring** |
|  | * Solder three motor wires from each motor to the three output pads of its ESC * Motor direction can be reversed later by swapping any two of the three wires * Route all ESC power leads to the power distribution board (PDB) * Solder ESC power leads to PDB: RED to positive; BLACK to negative * Zip-tie all wiring flat against frame arms to prevent vibration and entanglement |

|  |  |
| --- | --- |
| **STEP 5** | **Flight Controller Mounting & Connection** |
|  | * Mount FC on central plate using anti-vibration standoffs (rubber dampers) * Arrow or N mark on FC must point toward the front of the drone * Connect ESC signal wires to FC motor outputs: M1 (Front-Left), M2 (Front-Right), M3 (Rear-Right), M4 (Rear-Left) * Connect receiver to FC UART port using the supplied cable |

**Lessons 5–6 – Configuration & Pre-Flight**

|  |  |
| --- | --- |
| **STEP 6** | **Flight Controller Configuration (Betaflight)** |
|  | * Connect FC to computer via USB; open Betaflight Configurator * Set firmware to match your FC board; enable DSHOT300 protocol for ESCs * Configure motor layout: X configuration for standard quadcopter * Calibrate accelerometer with drone on a level surface * Set receiver protocol to match transmitter (SBUS, IBUS, or CRSF) |

|  |  |
| --- | --- |
| **STEP 7** | **Radio Binding & Control Verification** |
|  | * Put transmitter in bind mode; power on receiver; complete binding procedure * In Betaflight Receiver tab: move each stick; confirm correct channel responds * Verify directions: throttle up = channel value increases * Set failsafe: motors disarm if signal lost for more than 0.5 seconds |

|  |  |
| --- | --- |
| **STEP 8** | **Motor Direction Check – WITHOUT PROPELLERS** |
|  | * REMOVE ALL PROPELLERS BEFORE THIS STEP * In Betaflight Motor tab: spin each motor individually at 10% throttle * Confirm each motor spins in correct direction per Motor Configuration table * If a motor spins wrong: swap any two of its three ESC-to-motor wires * Repeat until ALL four motors spin correctly * Only fit propellers AFTER direction is confirmed |

**Lessons 7–8 – Pre-Flight Check & Tethered Flight**

|  |  |
| --- | --- |
| **STEP 9** | **Pre-Flight Inspection & Teacher Sign-Off** |
|  | * Complete the 10-point pre-flight checklist — every item must be ticked before flight * Teacher countersigns before any battery is connected * Move to designated outdoor flight zone * Anchor tether cord securely to the ground; attach other end to drone's central plate * Teacher connects LiPo battery and performs arm test |

|  |  |
| --- | --- |
| **STEP 10** | **Tethered Hover & Flight Log** |
|  | * Pilot stands at least 3 metres behind the drone * Slowly increase throttle until drone lifts off — target 1 m altitude * Hold hover for 10 seconds; reduce throttle to land * Teacher disarms motors; teacher disconnects battery before anyone approaches * Record one entry in flight log per flight; complete 5 tethered flights |

**4. Power & Safety Notes**

|  |
| --- |
| **⚠ CRITICAL SAFETY PROTOCOL**  LiPo batteries: Teacher handles all connections. Students must NOT connect or disconnect batteries.  Propeller guards: MANDATORY at all times. No flight permitted without guards.  Tether: First 5 flights MUST be tethered. No exceptions.  Flight zone: Minimum 10 m × 10 m clear area; no bystanders; no overhead obstacles.  Discharge: Land immediately if voltage falls below 3.5V per cell.  Damaged LiPo: Remove from service immediately; store in LiPo-safe bag.  NEVER put hands near motors while battery is connected. |

**5. Engineering Principles**

**Motor Configuration**

| **Motor #** | **Position** | **Spin Direction** | **Propeller Type** |
| --- | --- | --- | --- |
| **M1** | Front Left | Clockwise (CW) | CW threaded propeller |
| **M2** | Front Right | Counter-Clockwise (CCW) | CCW threaded propeller |
| **M3** | Rear Right | Clockwise (CW) | CW threaded propeller |
| **M4** | Rear Left | Counter-Clockwise (CCW) | CCW threaded propeller |

**Four Control Inputs**

| **Input** | **Stick** | **Motors Affected** | **Effect** |
| --- | --- | --- | --- |
| **Throttle** | Left – Up/Down | All 4 equally | Climb or descend |
| **Roll** | Right – Left/Right | Left pair up / Right pair down (or vice versa) | Bank left or right |
| **Pitch** | Right – Up/Down | Front pair up / Rear pair down (or vice versa) | Nose up or nose down |
| **Yaw** | Left – Left/Right | CW pair up / CCW pair down (or vice versa) | Rotate clockwise or counter-clockwise |

|  |
| --- |
| **Why Alternating CW/CCW Motors?**  Each spinning motor creates a torque reaction that would rotate the drone in the opposite direction.  Two CW + two CCW motors cancel the torques — the drone stays pointed forward.  To YAW: the FC increases CW pair speed and reduces CCW pair (or vice versa) — unbalanced torque rotates the drone. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Motors will not arm** | FC not configured or throttle not at minimum | Reconfigure FC; hold throttle to minimum for 3 seconds |
| **Drone flips on takeoff** | Wrong motor spin direction | Use Betaflight Motor tab to identify and correct; swap two motor wires |
| **Drone rotates (spins) on hover** | CW/CCW propellers swapped | Match CW prop to CW motor; CCW prop to CCW motor |
| **Battery discharges in < 5 min** | Cell damaged or over-discharged | Check cell voltage; replace if any cell < 3.5V under load |
| **No transmitter response** | Receiver not bound or failsafe triggered | Rebind: hold bind button while powering receiver; check failsafe settings |
| **Drone drifts sideways** | IMU not calibrated | Calibrate IMU in Betaflight with drone on a level surface |

**9. Upgrade & Extension Ideas**

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* Pre-solder motor-to-ESC connections for classes without soldering experience — focus time on assembly and configuration
* Install Betaflight Configurator before Lesson 5 — school networks may block executable downloads
* Designate one student per group as Safety Marshal — responsible for ensuring no one approaches while battery is connected
* First flight: teacher-operated transmitter with students observing before handing over control

**Differentiation Strategies**

* Support – Pre-assembled frame; students focus on electronics connection, Betaflight setup, and observation
* Core – Full build; all configuration; 5 tethered flights; complete flight log
* Extension – Altitude hold; FPV; GPS position hold; waypoint mission
