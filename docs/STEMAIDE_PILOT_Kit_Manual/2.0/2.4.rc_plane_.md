**Student & Teacher Manual**

**1. Project Overview**

This project develops precision RC piloting skills through the figure-8 circuit — the benchmark manoeuvre that tests coordinated turning, altitude management, speed control, and situational awareness simultaneously. Using the aircraft built in Project 11, students practise in the RC simulator before transitioning to the real aircraft. A structured scoring system rewards consistency and precision, not just completion.

**Learning Objectives**

* Describe the control inputs required for a coordinated banked turn (aileron + elevator + rudder)
* Explain the concept of wind correction and crab angle for maintaining intended ground track
* Complete a figure-8 circuit comprising two symmetrical 180° turns with a centre crossover
* Maintain altitude within ±2 m throughout the figure-8 pattern
* Record and score 5 flight attempts; identify the specific area requiring greatest improvement

**3. Session Steps**

**Lesson 1 – Simulator Refresher & Course Setup**

|  |  |
| --- | --- |
| **STEP 1** | **Simulator Figure-8 Practice** |
|  | * Before flying the real aircraft: complete 3 successful figure-8 patterns in the simulator * Use the same simulator from Project 4 (PicaSim, DroneSim, or RealFlight) * Focus on: adding elevator in every turn; starting rollout before the exit point * Only students who complete 3 simulator patterns may fly the real aircraft |

|  |  |
| --- | --- |
| **STEP 2** | **Course Setup** |
|  | * Find an open field with at least 80 m × 40 m of clear space * Place two cones 30 m apart along the centre line — these are the pylon markers * Mark the crossover centre point with a third cone between the pylons * Place a fourth cone at the takeoff/landing point, oriented into the wind * Place a wind indicator (ribbon on stake) upwind of the course * Walk the course and identify any obstacles, slopes, or hazards |

**Lesson 2 – Circuit Familiarisation Flights**

|  |  |
| --- | --- |
| **STEP 3** | **Pre-Flight Checks** |
|  | * Complete the full pre-flight checklist from Project 11 * Teacher connects battery; pilot confirms all controls respond correctly * Check wind direction: plan circuit so takeoff is into wind * Establish safety zone: all observers remain behind the pilot at all times |

|  |  |
| --- | --- |
| **STEP 4** | **Practice Flights – No Scoring** |
|  | * Flights 1 and 2 are practice only — no scoring applied * Focus: fly over the two pylons in a figure-8 pattern; do not worry about precision yet * After each flight: pilot and teacher debrief — what went well? Where was altitude lost? * Identify the strongest error from the Technique Table (Section 5) and focus on fixing it |

**Lessons 3–4 – Scored Runs & Competition**

|  |  |
| --- | --- |
| **STEP 5** | **Scored Runs** |
|  | * Flights 3, 4, and 5 are scored using the Scoring Table (Section 6) * Observer (teacher or assigned student) scores each category during the flight * Pilot lands after completing 3 full figure-8 patterns — or after a crash, stall, or airspace departure * Record score in flight log immediately after landing; discuss with pilot |

|  |  |
| --- | --- |
| **STEP 6** | **Improvement Focus Flight** |
|  | * After 3 scored runs, each pilot identifies their lowest-scoring category * Run 4 is a focused practice flight: attempt to improve ONLY the identified weak area * This mirrors real pilot training: targeted practice on the weakest skill |

|  |  |
| --- | --- |
| **STEP 7** | **Competition Final & Debrief** |
|  | * Each pilot's best scored run is used for the competition ranking * Award titles: Highest Total Score, Best Landing, Smoothest Turns * Class debrief: What was the hardest part? What would you practise first next time? |

**5. Engineering Principles**

**Figure-8 Technique Guide**

| **Manoeuvre** | **Inputs Required** | **Common Error** | **Correction** |
| --- | --- | --- | --- |
| **Entry to turn** | Aileron to bank; elevator to maintain altitude | No elevator — nose drops | Add back pressure as you bank |
| **Holding the turn** | Sustained aileron; continuous elevator | Over-banking; losing height | Reduce bank; add more elevator |
| **Rollout** | Opposite aileron; reduce elevator | Overshooting the exit heading | Start rollout 10° before target heading |
| **Crossover (centre)** | Level wings; check altitude | Too high or too low at crossover | Correct throttle on downwind leg before crossover |
| **Wind correction** | Crab angle into wind | Drift off intended track | Point nose slightly into wind; hold the heading |

|  |
| --- |
| **The Coordinated Turn – Key Concept**  A banked turn requires THREE simultaneous inputs: aileron (to bank), elevator (to maintain altitude), and slight rudder (to prevent adverse yaw).  In a 30° banked turn, 15% of the wing's lift is being used to turn rather than support weight — add elevator to compensate.  The steeper the bank, the more elevator is needed. A 60° bank requires twice the lift of level flight.  Failing to add elevator is the most common beginner error — the nose drops and altitude is lost. |

**Wind Correction**

* Wind pushes the aircraft sideways relative to the intended ground track
* Correction: point the nose slightly into wind (crab angle) so sideways drift cancels the crosswind
* Into-wind turns are tighter (less ground distance); downwind turns are wider
* Plan circuit geometry with the wind in mind: fly tighter patterns on into-wind legs

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Loses altitude in turns** | Insufficient elevator in banked turn | Add back pressure as soon as bank begins; practise in simulator first |
| **Overshoots turn exits** | Speed too high entering the turn | Reduce throttle before turn; start rollout 10° early |
| **Stalls in tight turn** | Speed too low in banked turn | Increase throttle in turns; maintain safe airspeed throughout |
| **Drift off course in wind** | No crab angle applied | Point nose into wind; maintain intended track over ground |
| **Crossover point inconsistent** | Throttle not steady on straight legs | Set a fixed throttle setting; check altitude at crossover each time |

**9. Upgrade & Extension Ideas**

* Timed Challenge – Score both accuracy AND time; fastest correct pattern wins
* Vertical Figure-8 – For advanced pilots: fly the pattern in the vertical plane (inside loops)
* Obstacle Course – Add ribbon gates at each pylon that pilots must fly through precisely
* FPV Pattern – Fly the figure-8 in first-person view if FPV equipment is available
* Video Analysis – Record flights from the ground; review footage to identify turn entry and exit points precisely

**10. Teacher Notes & Differentiation**

**Lesson Planning Tips**

* The simulator requirement in Lesson 1 is not optional — students who skip it almost always crash on the first real attempt
* Brief student observers on how to score fairly before flights begin — run through one example flight as a class
* Wind above 15 km/h makes this exercise very difficult for beginners — reschedule if necessary
* Have spare propellers and repair tape ready — minor crashes are expected and the aircraft should be quickly repairable

**Differentiation Strategies**

* Support – Complete 2 patterns instead of 3; no altitude consistency requirement; focus on turn quality only
* Core – 3 patterns; full scoring table; 5 flights logged; improvement focus flight
* Extension – Timed challenge; FPV system; video analysis; vertical figure-8
