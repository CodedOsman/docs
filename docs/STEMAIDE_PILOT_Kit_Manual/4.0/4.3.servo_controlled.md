**Student & Teacher Manual**

**1. Project Overview**

This project builds the physical understanding of how control surfaces work before students install them on a real flying aircraft. By constructing a working aileron model with a micro servo, pushrod, and control horn, then building elevator and rudder demonstrations, students develop both the mechanical skills and the conceptual understanding needed for Project 11. The demonstrator also serves as a permanent teaching aid for the classroom.

**Learning Objectives**

* Name the three primary flight control surfaces and their axis of rotation (aileron/roll, elevator/pitch, rudder/yaw)
* Assemble a servo-to-control-horn linkage using a pushrod with Z-bend ends
* Verify correct control direction for all three surfaces: aileron, elevator, and rudder
* Explain why incorrect control direction causes an immediate crash on a real aircraft
* Demonstrate each control surface to a partner while explaining the aircraft effect
* Use the demonstrator as a reference when installing servos in Project 11

**3. Build Steps & Assembly**

**Lesson 1 – Aileron Demonstrator**

|  |  |
| --- | --- |
| **STEP 1** | **Build the Wing Section** |
|  | * Cut two foam board pieces: main wing panel (25 cm × 10 cm) and aileron panel (10 cm × 3 cm) * Score (do not cut through) the foam along the hinge line — creates a flexible foam hinge * Reinforce the hinge with a 10 cm strip of masking tape on the upper surface * Test the hinge: the aileron should deflect 20 mm up and 20 mm down freely with no tearing |

|  |  |
| --- | --- |
| **STEP 2** | **Servo Installation** |
|  | * Cut a servo pocket in the main wing panel using the servo as a template * Insert servo so the output arm aligns with the hinge line * Hot glue the servo in place on three sides; leave one side accessible * Glue the control horn to the aileron panel, centred and perpendicular to the hinge |

|  |  |
| --- | --- |
| **STEP 3** | **Pushrod Assembly & Direction Check** |
|  | * Bend a Z-bend at one end of the pushrod; insert into the outermost hole of the control horn * Connect the other end to the servo output arm * Connect servo lead to battery; centre the servo * Move servo arm forward: aileron should deflect downward; backward: upward * If direction is wrong: move servo arm to the opposite side of the output shaft * Verify against the Control Reference Table (Section 5) |

**Lessons 2–3 – Elevator & Rudder**

|  |  |
| --- | --- |
| **STEP 4** | **Build Elevator Demonstrator** |
|  | * Cut horizontal stabilizer (20 cm × 8 cm) and elevator (20 cm × 2.5 cm) * Score and tape the hinge; install servo and control horn using the same method as Steps 1–3 * Verify direction: right stick backward → elevator trailing edge UP (nose pitches up) |

|  |  |
| --- | --- |
| **STEP 5** | **Build Rudder Demonstrator** |
|  | * Cut vertical fin (15 cm × 8 cm) and rudder (15 cm × 2.5 cm) * Score and tape the hinge vertically; install servo and horn * Verify direction: left stick right → rudder deflects right (nose yaws right) |

|  |  |
| --- | --- |
| **STEP 6** | **Class Demonstration** |
|  | * Mount all three demonstrators on a display board with the aircraft axes labelled * Each student takes turns operating the controls and naming: the surface, the axis, and the aircraft effect * Teacher uses the demonstrators as a reference point during Project 11 servo installation |

**5. Engineering Principles**

**Control Surface Reference Table**

| **Surface** | **Input** | **Movement** | **Aircraft Effect** |
| --- | --- | --- | --- |
| **Aileron** | Left stick right | Left aileron UP; Right aileron DOWN | Roll right |
| **Aileron** | Left stick left | Left aileron DOWN; Right aileron UP | Roll left |
| **Elevator** | Right stick back | Trailing edge UP | Pitch nose up |
| **Elevator** | Right stick forward | Trailing edge DOWN | Pitch nose down |
| **Rudder** | Left stick right | Trailing edge RIGHT | Yaw nose right |
| **Rudder** | Left stick left | Trailing edge LEFT | Yaw nose left |

* Servo mechanics: a servo motor drives a plastic output arm to a precise angle based on a PWM (pulse-width modulation) signal from the receiver
* Pushrod: transfers servo arm motion to the control surface horn; must be rigid to avoid backlash and flex under load
* Control horn: lever arm on the control surface; the further from the hinge, the more mechanical advantage but less throw
* Hinge: must allow free rotation without friction — any binding reduces control authority and can lead to control surface flutter

|  |
| --- |
| **Why Control Direction Matters**  A reversed elevator is one of the most common causes of first-flight crashes in RC aviation.  When the pilot pushes forward to correct a nose-up attitude, a reversed elevator pitches the nose further up — into an unrecoverable stall.  All aircraft manufacturers verify control direction through a mandatory pre-flight check before every flight.  The verification sequence from Project 5 (Pre-Flight Checklist) is the standard solution to this problem. |

**7. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **Servo does not move** | Wrong wiring: signal, power, or ground swapped | Check servo lead: orange/yellow = signal; red = power; brown/black = ground |
| **Control surface binds (won't move fully)** | Hinge too tight or pushrod misaligned | Loosen hinge tape; realign pushrod so it pulls in a straight line |
| **Servo moves but surface doesn't** | Pushrod disconnected from control horn | Re-insert Z-bend into outermost horn hole; check clevis is locked |
| **Movement in wrong direction** | Servo arm on wrong side | Move servo arm to opposite side; or reverse servo direction in transmitter |
| **Servo jitters constantly** | Low battery voltage | Replace or recharge the 4.8V battery |

**9. Upgrade & Extension Ideas**

* Radio Control: connect the demonstrators to the Project 11 RC transmitter and receiver; fly the surfaces from across the room
* Multiple Servos: add a second aileron servo for differential aileron control; observe the difference from a single-servo setup
* Deflection Measurement: use a protractor to measure deflection angle at different servo arm positions; calculate the relationship between arm length and throw
* Flaps: add a hinged flap section to the trailing inner wing; demonstrate the lift increase by holding the panel in an airstream

**10. Teacher Notes & Differentiation**

* Build one complete demonstration unit before Lesson 1 — students need to see the finished product to understand what they are building
* The direction check (Step 3) is the most important step — dedicate 10 minutes to it; make every student verify all three surfaces independently
* Support – Pre-scored foam panels; focus on servo connection and direction verification only
* Core – Full build of all three demonstrators; control reference table completed; class demonstration
* Extension – RC connection; deflection measurement; flap addition
