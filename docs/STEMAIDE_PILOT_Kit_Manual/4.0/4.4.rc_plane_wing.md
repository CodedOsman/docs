**Student & Teacher Manual**

**1. Project Overview**

Wing loading — the ratio of aircraft weight to wing area — is one of the most fundamental parameters in aircraft design. Low wing loading means slow, maneuverable, forgiving flight; high wing loading means fast, stable flight but a higher stall speed and reduced tolerance for pilot error. This project gives students direct, first-hand experience of how wing loading changes the character of an RC aircraft, using the plane they built in Project 11 as a controlled test subject.

**Learning Objectives**

* Define wing loading (weight ÷ wing area) and explain its units (g/cm² or kg/m²)
* Calculate the wing area of the Project 11 aircraft and compute baseline wing loading
* Fly five weight configurations (0 g, 10 g, 20 g, 30 g, 40 g added) and record stall speed and handling
* Plot a graph of wing loading vs. estimated stall speed
* Identify the maximum safe wing loading from the data and explain the reasoning
* Compare wing loading values for different real aircraft types (glider, trainer, cargo)

**3. Lesson Guide & Steps**

**Lesson 1 – Calculations & Pre-Flight**

|  |  |
| --- | --- |
| **STEP 1** | **Baseline Measurements** |
|  | * Weigh the aircraft without any added weights; record in grams * Measure wing span (tip to tip) and average chord (width from leading to trailing edge) * Calculate wing area: Span (cm) × Chord (cm) = Area (cm²) * Calculate baseline wing loading: Aircraft Weight (g) ÷ Wing Area (cm²) * Mark the CG point on the fuselage — all added weights must go here |

|  |  |
| --- | --- |
| **STEP 2** | **Weight Preparation** |
|  | * Prepare 5 × 10 g weights: small coin, measured clay blob, or washers * Verify each weight on the scale; label them W1 through W5 * Test the mounting method: tape a weight to the CG point; check the aircraft still balances * Confirm the tape method holds securely in a 3-second vigorous shake test |

**Lessons 2–3 – Flight Testing**

|  |  |
| --- | --- |
| **STEP 3** | **Flight Test Protocol** |
|  | * Complete the full pre-flight checklist from Project 11 before every test configuration * Teacher connects battery; pilot completes a 60-second evaluation circuit * During flight: note stall onset (any mushing, dropping nose, loss of altitude control at low speed) * Estimate the minimum speed before stall; assign a handling score (5 = very responsive, 1 = barely controllable) * Land; teacher disconnects battery; add the next weight increment; repeat |

|  |  |
| --- | --- |
| **STEP 4** | **Safety Stop Criteria** |
|  | * If handling score drops to 2 or below: STOP adding weight; this is the maximum safe loading * If the aircraft stalls at circuit altitude without warning: STOP; this configuration is unsafe * Record the maximum safe total weight and calculate the maximum safe wing loading |

**Lesson 4 – Analysis**

|  |  |
| --- | --- |
| **STEP 5** | **Graph & Analysis** |
|  | * Plot wing loading (g/cm²) on the x-axis; estimated stall speed on the y-axis * Add a second y-axis for handling score; plot it as a dashed line * Identify the crossover point where performance degrades unacceptably * Research real aircraft: a glider has ~15 g/m² wing loading; a Cessna 172 has ~70 kg/m²; a fighter jet > 600 kg/m² * Write a 2-paragraph analysis comparing your aircraft data to the real-world examples |

**4. Power & Safety Notes**

|  |
| --- |
| **⚠ Safety Requirements**  Power: LiPo battery – teacher connects and disconnects only.  Weight placement: All weights MUST be secured at the CG point. Weight elsewhere changes CG and makes the aircraft unpredictable.  Abort criteria: If handling score drops to 2 or the aircraft stalls unexpectedly, immediately stop the experiment.  Flight zone: Designated outdoor area; all observers behind the pilot.  Do not fly with 40 g added weight if the aircraft was marginal with 30 g — safety takes priority over data completeness. |

**5. Engineering Principles**

* Wing loading = Total Weight ÷ Wing Area. Units: g/cm² (model aircraft) or kg/m² (full-size aircraft)
* Stall speed increases with wing loading: to generate the same lift with more weight, the aircraft must fly faster
* Low wing loading: slow stall speed, forgiving handling, slow cruise — gliders, sailplanes, trainers
* High wing loading: fast stall speed, stable in turbulence, fast cruise — cargo aircraft, fighters, airliners

|  |
| --- |
| **Stall Speed Formula**  Stall speed increases with the square root of wing loading.  If wing loading doubles, stall speed increases by √2 ≈ 1.41 times.  Example: baseline stall speed 20 km/h; double the wing loading → stall speed = 28 km/h.  This is why heavily loaded cargo aircraft need much longer runways for takeoff and landing. |

**8. Common Errors & Fixes**

| **Error** | **Likely Cause** | **Fix** |
| --- | --- | --- |
| **CG shifts with added weight** | Weights not placed at CG point | Tape all weights at the marked CG position; re-check balance before each flight |
| **Stall speed not identifiable** | Aircraft too fast to observe stall onset | Reduce throttle gradually during level flight; note the speed at which the aircraft begins to mush |
| **Handling score very subjective** | No consistent criteria | Define specific handling criteria before the first flight: e.g. '5 = responds in < 0.5 s; 1 = barely controllable' |
| **Plane crashes with 30 g weight** | Exceeded safe limit | Stop adding weight; fly no further; identify the highest safe loading from previous data |
| **Inconsistent results** | Different wind conditions between tests | Fly all tests on the same day within a 30-minute window; note wind direction and speed |

**9. Upgrade & Extension Ideas**

* Glide Ratio: measure glide distance at each weight increment with motor off; graph wing loading vs. glide ratio
* Climb Rate: time the climb to 20 m altitude at each weight; graph wing loading vs. climb rate
* Wind Effect: fly the same weight configuration in 5 km/h and 10 km/h wind; compare handling scores
* Real Aircraft Comparison: calculate wing loading for a Boeing 737 and your RC plane; produce a comparison chart

**10. Teacher Notes & Differentiation**

* The experiment requires the aircraft to be already airworthy — verify this before scheduling the lesson
* For the stall speed estimate: ask the pilot to gradually reduce throttle during level flight until the aircraft begins to descend despite up elevator — that is stall onset
* Support – 3 weight steps only (0, 20, 40 g); pre-calculated wing area; focus on observation and handling score only
* Core – All 5 steps; full calculation; graph; real aircraft comparison
* Extension – Glide ratio; climb rate; wind effect comparison
