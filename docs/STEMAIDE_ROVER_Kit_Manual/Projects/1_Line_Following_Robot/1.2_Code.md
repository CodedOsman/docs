# 1.2 Line Following Program Block by Block

## 1. Start Block

**1.** Drag the block **"when Arduino Uno starts up"**.

![Image 74](../../../../assets/rover/pictoblox-manual/image_074.png)

**2.** Attach the **"forever"** loop block. Everything below will go inside this forever block.

![Image 75](../../../../assets/rover/pictoblox-manual/image_075.png)

## 2. Move Forward (Both Sensors on White)

**1.** Add an **"if"** block:
`if (read analog pin A3 = 0) and (read analog pin A2 = 0) then`

![Image 76](../../../../assets/rover/pictoblox-manual/image_076.png)
![Image 77](../../../../assets/rover/pictoblox-manual/image_077.png)
![Image 78](../../../../assets/rover/pictoblox-manual/image_078.png)
![Image 79](../../../../assets/rover/pictoblox-manual/image_079.png)

**2.** Inside the block, place the motor commands:
- `run motor 2 forward with speed 90%`
- `run motor 3 forward with speed 90%`

The robot moves **straight forward**.

![Image 80](../../../../assets/rover/pictoblox-manual/image_080.png)

## 3. Turn Right (Left Sensor on Line)

**1.** Add an **"else if"** condition:
`if (read analog pin A3 = 0) and (read analog pin A2 = 1) then`

![Image 81](../../../../assets/rover/pictoblox-manual/image_081.png)

**2.** Inside it, place:
- `run motor 2 forward with speed 0%`
- `run motor 3 forward with speed 90%`

The robot **turns right**.

![Image 82](../../../../assets/rover/pictoblox-manual/image_082.png)

## 4. Turn Left (Right Sensor on Line)

**1.** Add another **"else if"** condition:
`if (read analog pin A3 = 1) and (read analog pin A2 = 0) then`

![Image 83](../../../../assets/rover/pictoblox-manual/image_083.png)

**2.** Inside it, place:
- `run motor 2 forward with speed 90%`
- `run motor 3 forward with speed 0%`

The robot **turns left**.

![Image 84](../../../../assets/rover/pictoblox-manual/image_084.png)

## 5. Stop (Both Sensors on Black)

**1.** Add the final **"if"** block:
`if (read analog pin A3 = 1) and (read analog pin A2 = 1) then`

![Image 85](../../../../assets/rover/pictoblox-manual/image_085.png)
![Image 86](../../../../assets/rover/pictoblox-manual/image_086.png)

**2.** Inside it, place:
- `run motor 2 forward with speed 0%`
- `run motor 3 forward with speed 0%`

The robot **stops**.

![Image 87](../../../../assets/rover/pictoblox-manual/image_087.png)
