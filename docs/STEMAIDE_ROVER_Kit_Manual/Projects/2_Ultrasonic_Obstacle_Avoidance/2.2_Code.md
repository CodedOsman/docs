# 2.2 Ultrasonic Obstacle Avoidance Program Block by Block

## 1. Start the Program

**1.** Drag the block **"when Arduino Uno starts up"**.

![Image 91](../../../../assets/rover/pictoblox-manual/image_091.png)

**2.** Attach the **"forever"** loop. Everything will go inside this forever loop.

![Image 92](../../../../assets/rover/pictoblox-manual/image_092.png)

## 2. Read the Distance

**1.** Inside the loop, add a **set variable** block. Create a variable called **distance**.
**2.** Configure it: `set distance to ultrasonic sensor reading (Trig A4 Echo A5)`

This lets the robot know **how far the obstacle is**.

![Image 93](../../../../assets/rover/pictoblox-manual/image_093.png)
![Image 94](../../../../assets/rover/pictoblox-manual/image_094.png)

## 3. Add an IF Block

**1.** Inside the forever block, add an **IF** block:
`if ultrasonic sensor distance < 20 cm then`

This triggers when an object is **closer than 20 cm**.

![Image 95](../../../../assets/rover/pictoblox-manual/image_095.png)
![Image 96](../../../../assets/rover/pictoblox-manual/image_096.png)

## 4. Stop the Robot

Inside the **IF** block, add:
- `run motor 2 forward speed 0%`
- `run motor 3 forward speed 0%`

The robot **stops**.

![Image 97](../../../../assets/rover/pictoblox-manual/image_097.png)

## 5. Move Backward

Still inside the **IF** block, add:
- `run motor 2 backward speed 50%`
- `run motor 3 backward speed 50%`
- `wait 1 seconds`

The robot **moves away from the obstacle**.

![Image 98](../../../../assets/rover/pictoblox-manual/image_098.png)

## 6. Turn the Robot

Still inside the **IF** block, add:
- `run motor 2 forward speed 80%`
- `run motor 3 backward speed 80%`
- `wait 1 seconds`

The robot **turns to avoid the obstacle**.

![Image 99](../../../../assets/rover/pictoblox-manual/image_099.png)

## 7. Move Forward When Path Is Clear

Now add an **ELSE** clause. Inside **ELSE**, add:
- `run motor 2 forward speed 90%`
- `run motor 3 forward speed 90%`

The robot **moves forward** when there are no obstacles.

![Image 100](../../../../assets/rover/pictoblox-manual/image_100.png)

## Final Simple Logic

1. **Start**
2. **Forever Loop**
3. **If obstacle < 20 cm**
    - Stop
    - Go backward
    - Turn
4. **Else**
    - Move forward

![Image 101](../../../../assets/rover/pictoblox-manual/image_101.png)

That is the **full obstacle-avoiding robot program**.
