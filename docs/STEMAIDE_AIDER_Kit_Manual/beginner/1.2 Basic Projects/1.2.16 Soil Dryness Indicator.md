# Soil Dryness Indicator

# Overview

Build a soil dryness monitor that warns when soil moisture is too low.

This project combines an analog moisture sensor, a buzzer alert, an OLED display, and a local web page.

The final result should show a moisture percentage, sound an alert when the soil is too dry, and display the same status in a browser.

# Full Project Code

Upload and run this code after the individual tests work correctly.

| import network
import socket
import time
from machine import ADC, I2C, Pin
import sh1106

SSID = 'YOUR_WIFI_NAME'
PASSWORD = 'YOUR_WIFI_PASSWORD'

adc = ADC(Pin(26))
DRY_VALUE = 58000
WET_VALUE = 20000
THRESHOLD = 30

buzzer = Pin(0, Pin.OUT)

i2c = I2C(0, sda=Pin(8), scl=Pin(9), freq=400000)
oled = sh1106.SH1106_I2C(128, 64, i2c)


def moisture_percent():
    raw = adc.read_u16()
    if raw >= DRY_VALUE:
        return raw, 0
    if raw <= WET_VALUE:
        return raw, 100
    percent = int((DRY_VALUE - raw) / (DRY_VALUE - WET_VALUE) * 100)
    return raw, percent


def status_text(percent):
    if percent < THRESHOLD:
        return 'DRY - WATER NOW!'
    return 'OK'


def web_page(percent, status):
    return '''<!DOCTYPE html>
<html>
<head>
    <meta name='viewport' content='width=device-width, initial-scale=1'>
    <meta http-equiv='refresh' content='3'>
    <title>Soil Dryness Indicator</title>
</head>
<body style='font-family:Arial;text-align:center;padding:30px'>
    <h1>Soil Dryness Indicator</h1>
    <h2>{}%</h2>
    <p>{}</p>
    <p>Alert threshold: {}%</p>
</body>
</html>'''.format(percent, status, THRESHOLD)


wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

print('Connecting to Wi-Fi...')
for _ in range(15):
    if wlan.isconnected():
        break
    time.sleep(1)

if not wlan.isconnected():
    raise RuntimeError('Wi-Fi connection failed')

ip_address = wlan.ifconfig()[0]
print('Connected. Open http://{} in your browser'.format(ip_address))

address = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
server = socket.socket()
server.bind(address)
server.listen(1)
server.settimeout(0.2)

while True:
    raw, percent = moisture_percent()
    status = status_text(percent)
    buzzer.value(1 if percent < THRESHOLD else 0)

    oled.fill(0)
    oled.text('Soil Moisture', 8, 8, 1)
    oled.text('{}%'.format(percent), 44, 28, 1)
    oled.text(status[:16], 0, 50, 1)
    oled.show()

    try:
        client, client_address = server.accept()
    except OSError:
        continue

    client.recv(1024)
    response = web_page(percent, status)
    client.send('HTTP/1.1 200 OK\r\nContent-Type: text/html\r\nConnection: close\r\n\r\n'.encode())
    client.sendall(response.encode())
    client.close() |

# How the Code Works

| Code Section | What It Does | Why It Matters |
| --- | --- | --- |
| Calibration values | Store dry and wet ADC readings | The soil sensor needs calibration before the percentage makes sense |
| Inverted calculation | Turns lower wet readings into higher moisture percentages | Many soil sensors behave in the opposite way beginners expect |
| Threshold check | Decides when the buzzer should sound | This turns the monitor into an alert system |
| OLED and web output | Show the same dryness status in two places | Students can compare local and remote display results |

# Expected Result

After entering your Wi-Fi details and running the code, the OLED and browser page should show the soil moisture percentage. When the moisture drops below the threshold, the buzzer should sound and the status should change to a dry warning.

# Troubleshooting

| Problem | Possible Cause | Solution |
| --- | --- | --- |
| Always shows 0% | Calibration values are wrong or sensor is disconnected | Print raw ADC values and recheck DRY_VALUE, WET_VALUE, and wiring |
| Always shows 100% | DRY_VALUE and WET_VALUE may be reversed | Swap or recalibrate the values based on your sensor |
| Buzzer never sounds | Threshold is too low or buzzer wiring is wrong | Raise the threshold for testing and recheck the buzzer |
| Sensor corrodes over time | This is common with resistive soil probes | Remove the sensor when not needed or switch to a capacitive sensor for long-term use |
