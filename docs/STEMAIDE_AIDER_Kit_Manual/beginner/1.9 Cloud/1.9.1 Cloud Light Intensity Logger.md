## Cloud Light Intensity Logger

# Overview

Build a light logger that reads an LDR module and shows the current value plus recent history on a web page.

This project demonstrates analog input, a rolling history list, and browser-based data display.

In this beginner version, the data is shown on a web page on your local Wi-Fi network. The final result should show the current light level, the last 10 readings, and their average.

# Full Project Code

Upload and run this code after the individual tests work correctly.

```python
import network
import socket
import time
from machine import ADC, Pin

SSID = 'YOUR_WIFI_NAME'
PASSWORD = 'YOUR_WIFI_PASSWORD'

adc = ADC(Pin(26))
light_log = []

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

print('Connecting to Wi-Fi...')
for _ in range(15):
    if wlan.isconnected():
        break
    time.sleep(1)

if not wlan.isconnected():
    raise RuntimeError('Wi-Fi connection failed')

ip_address = wlan.ifconfig()[0]
print('Connected. Open http://{} in your browser'.format(ip_address))

def read_light_percentage():
    raw = adc.read_u16()
    percent = int((raw / 65535) * 100)
    return raw, percent

def web_page(current_raw, current_percent, history):
    rows = ''
    for index, value in enumerate(history, 1):
        rows += '<tr><td>{}</td><td>{}%</td></tr>'.format(index, value)
    average = sum(history) / len(history) if history else 0
    return """<!DOCTYPE html>
<html>
<head>
    <meta name='viewport' content='width=device-width, initial-scale=1'>
    <meta http-equiv='refresh' content='3'>
    <title>Light Logger</title>
</head>
<body style='font-family:Arial;padding:30px'>
    <h1>Light Intensity Logger</h1>
    <h2>Current light: CURRENT_PERCENT%</h2>
    <p>Raw ADC: CURRENT_RAW</p>
    <h3>Last RECENT_COUNT readings</h3>
    <table border='1' cellpadding='8' cellspacing='0'>
        <tr><th>#</th><th>Light %</th></tr>
        HISTORY_ROWS
    </table>
    <p>Average: AVG_VALUE%</p>
    <p>Page refreshes every 3 seconds</p>
</body>
</html>""".replace('CURRENT_PERCENT', str(current_percent)).replace('CURRENT_RAW', str(current_raw)).replace('RECENT_COUNT', str(len(history))).replace('HISTORY_ROWS', rows).replace('AVG_VALUE', '{:.1f}'.format(average))

address = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
server = socket.socket()
server.bind(address)
server.listen(1)

while True:
    client, client_address = server.accept()
    print('Client connected from', client_address)
    client.recv(1024)

    current_raw, current_percent = read_light_percentage()
    light_log.append(current_percent)
    if len(light_log) > 10:
        light_log.pop(0)

    response = web_page(current_raw, current_percent, light_log)
    client.send('HTTP/1.1 200 OK\r\nContent-Type: text/html\r\nConnection: close\r\n\r\n'.encode())
    client.sendall(response.encode())
    client.close()
```

# How the Code Works

| Code Section | What It Does | Why It Matters |
| --- | --- | --- |
| Voltage divider + ADC | Reads the LDR signal as a 16-bit number | The Pico needs ADC to measure changing light levels |
| read_light_percentage() | Converts the raw value to a simple percentage | Percent values are easier for beginners to understand |
| light_log list | Stores only the last 10 readings | This creates a rolling history instead of an endlessly growing list |
| web_page() | Builds the HTML page with the current value, history, and average | The browser page becomes the simple data dashboard |

# Expected Result

After entering your Wi-Fi details and running the code, the Shell should print an IP address. Opening that address in a browser should show the current light level, a table of the last 10 readings, and the average. Covering or lighting the LDR should change the values.
