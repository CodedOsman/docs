# Project 1.4.3: Relay Timer Switch

---

## Overview

Build a relay timer switch with a button and an LED indicator.

This project demonstrates timed automation and safe relay control.

The final result should turn the relay and indicator LED on for a set time and then turn them off automatically.

## Full Project Code

```python
from machine import Pin
import time

relay = Pin(0, Pin.OUT)
button = Pin(1, Pin.IN, Pin.PULL_UP)
led = Pin(2, Pin.OUT)
RUN_TIME = 10

relay.value(1)
led.off()

print('Relay timer switch ready')
print('Press the button to start the timer')

while True:
    if button.value() == 0:
        print('Timer started')
        relay.value(0)
        led.on()

        for remaining in range(RUN_TIME, 0, -1):
            print('Time remaining:', remaining, 's')
            time.sleep(1)

        relay.value(1)
        led.off()
        print('Timer complete')
        time.sleep(0.3)

    time.sleep(0.02)
```

## How the Code Works

| Code Section     | What It Does                                   | Why It Matters                |
| ---------------- | ---------------------------------------------- | ----------------------------- |
| `RUN_TIME`       | Stores how long the relay stays active         | Timer can be changed easily   |
| Button input     | Starts the timer when pressed                  | Gives manual control          |
| Relay and LED on | Turns on relay and indicator LED               | Shows when timer is active    |
| Countdown loop   | Prints remaining seconds and turns outputs off | Creates timed switch behavior |

---

## Expected Result

Pressing the button turns on the relay and LED for about 10 seconds. After the countdown finishes, both turn off automatically.

## Troubleshooting

| Problem                     | Possible Cause                     | Solution                                |
| --------------------------- | ---------------------------------- | --------------------------------------- |
| Relay does not click        | Wrong relay power or logic         | Check VCC, GND, and active-low behavior |
| LED stays off               | LED reversed or resistor misplaced | Reverse LED and check resistor path     |
| Timer restarts unexpectedly | Button bouncing or held down       | Release button after one press          |
