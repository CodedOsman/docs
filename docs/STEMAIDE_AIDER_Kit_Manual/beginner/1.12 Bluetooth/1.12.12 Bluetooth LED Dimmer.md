# Overview

Build a Bluetooth LED dimmer that lets your phone control brightness.

This project demonstrates PWM, which changes how bright an LED looks by switching it very quickly.

The final result should let a phone send a brightness value from 0 to 100 and smoothly change the LED brightness.

Connect the LED long leg to one end of the 220Î© resistor.

# Full Project Code

Upload and run this code after the individual tests work correctly.

```python
from machine import Pin, PWM
import bluetooth
import time
from ble_uart import BLEUART

led = PWM(Pin(0))
led.freq(1000)

ble = bluetooth.BLE()
ble.active(True)
uart = BLEUART(ble, name='Pico-Dimmer')

brightness_percent = 0

def set_brightness(percent):
    global brightness_percent
    if percent < 0:
        percent = 0
    if percent > 100:
        percent = 100
    brightness_percent = percent
    duty = int((percent * 65535) / 100)
    led.duty_u16(duty)


def on_rx(data):
    command = data.decode('utf-8').strip().lower()
    print('Received command:', command)

    if command == 'status':
        uart.write(('Brightness: {}%\n'.format(brightness_percent)).encode())
    elif command == 'off':
        set_brightness(0)
        uart.write(b'Brightness: 0%\n')
    elif command == 'full':
        set_brightness(100)
        uart.write(b'Brightness: 100%\n')
    elif command == 'help':
        uart.write(b'Send 0 to 100, or use status, off, full, help\n')
    else:
        try:
            percent = int(command)
            set_brightness(percent)
            uart.write(('Brightness: {}%\n'.format(brightness_percent)).encode())
        except ValueError:
            uart.write(b'Unknown command. Send help.\n')

uart.on_rx(on_rx)
set_brightness(0)

print('Bluetooth LED dimmer ready')
print('Send a number from 0 to 100 for brightness')
print('Other commands: status, off, full, help')

while True:
    time.sleep(0.1)
```

# How the Code Works

| Code Section               | What It Does                                         | Why It Matters                                       |
| -------------------------- | ---------------------------------------------------- | ---------------------------------------------------- |
| PWM LED setup              | Creates a PWM output on GPIO 0                       | PWM is what makes brightness control possible        |
| `set_brightness()`         | Converts a percentage into a 16-bit duty value       | Students can think in 0 to 100 instead of 0 to 65535 |
| `on_rx()` callback         | Reads Bluetooth text commands and changes brightness | This links the phone app to the LED output           |
| status and helper commands | Report current brightness and support quick commands | Makes the project easier to test and understand      |

# Troubleshooting

| Problem                             | Possible Cause                                               | Solution                                                                                               |
| ----------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------ |
| LED does not light                  | LED is reversed, the resistor is missing, or wiring is wrong | Check LED polarity, the resistor, and GPIO 0 wiring                                                    |
| Brightness does not change smoothly | The command is outside 0 to 100 or the LED test failed       | Send a number between 0 and 100 and rerun the PWM LED test                                             |
| Phone cannot find Pico-Dimmer       | BLE helper files are missing or Bluetooth is not active      | Check that `ble_uart.py` and `ble_advertising.py` are saved on the Pico and rerun the advertising test |
