# Project 1.12.6: Bluetooth Water Pump Controller

**Beginner Embedded Systems Project Using Raspberry Pi Pico 2 W and MicroPython**


# Overview

Build a Bluetooth-controlled water pump using a relay and external power.

This project demonstrates safe short-range wireless control of a pump.

The final result should let the phone turn the pump on or off and request the current pump state.

- - Relay GND connects to GND

- - External pump supply positive connects to relay COM

# Full Project Code

Upload and run this code after the individual tests work correctly.

```python
from machine import Pin
import bluetooth
import time
from ble_uart import BLEUART

relay = Pin(0, Pin.OUT)
relay.value(1)

ble = bluetooth.BLE()
ble.active(True)
uart = BLEUART(ble, name='Pico-Pump')


def pump_on():
    relay.value(0)


def pump_off():
    relay.value(1)


def pump_state_text():
    return 'ON' if relay.value() == 0 else 'OFF'


def on_rx(data):
    command = data.decode('utf-8').strip().lower()
    print('Received command:', command)

    if command == 'on':
        pump_on()
        uart.write(b'Pump ON\n')
    elif command == 'off':
        pump_off()
        uart.write(b'Pump OFF\n')
    elif command == 'pulse':
        pump_on()
        time.sleep(2)
        pump_off()
        uart.write(b'Pump pulse complete\n')
    elif command == 'status':
        uart.write(('Pump status: ' + pump_state_text() + '\n').encode())
    elif command == 'help':
        uart.write(b'Commands: on, off, pulse, status, help\n')
    else:
        uart.write(b'Unknown command. Send help.\n')

uart.on_rx(on_rx)

print('Bluetooth water pump controller ready')
print('Connect with a BLE app and send commands: on, off, pulse, status, help')

while True:
    time.sleep(0.1)
```

# How the Code Works

| Code Section               | What It Does                              | Why It Matters                                                      |
| -------------------------- | ----------------------------------------- | ------------------------------------------------------------------- |
| Active-low relay setup     | Starts the pump relay in the OFF state    | This keeps the pump from starting unexpectedly                      |
| `pump_on()` / `pump_off()` | Wrap the relay control in clear functions | The code becomes easier to read and safer to update                 |
| `pulse` command            | Runs the pump for a short 2-second test   | This is a safer beginner test than leaving the pump on indefinitely |
| BLE command replies        | Send pump state feedback to the phone     | The user can confirm what happened after each command               |

---

# Expected Result

After running the code, your phone BLE app should find Pico-Pump. Sending `on` should turn the pump on through the relay. Sending `off` should turn it off. Sending `pulse` should run the pump briefly and then stop it. Sending `status` should report whether the pump is ON or OFF.

# Troubleshooting

| Problem                     | Possible Cause                                                 | Solution                                                   |
| --------------------------- | -------------------------------------------------------------- | ---------------------------------------------------------- |
| Pump does not run           | Relay wiring or external power is wrong                        | Check the pump supply, relay COM, relay NO, and pump wires |
| Pump never stops after on   | The off command was not sent or the relay is wired differently | Use the off command and verify active-low relay behavior   |
| Phone cannot find Pico-Pump | BLE libraries are missing or Bluetooth is not active           | Check the library files and rerun the advertising test     |
