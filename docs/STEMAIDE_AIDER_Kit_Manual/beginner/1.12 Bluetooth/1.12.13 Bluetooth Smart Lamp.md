**Beginner Embedded Systems Project Using Raspberry Pi Pico 2 W and MicroPython**


# Overview

Build a Bluetooth smart lamp that can turn on, turn off, and change brightness from a phone.

This project demonstrates how PWM can be used to make one LED behave like a simple dimmable lamp.

The final result should let a phone send commands such as `on`, `off`, `low`, `medium`, `high`, or a number from `0` to `100` to set the brightness.

## Step 3: Add the 220Î© Resistor

Connect the LED long leg to one end of the 220Î© resistor.

![Resistor Connection](../../../assets/aider/beginner/bluetooth/bluetooth_smart_lamp/circuit_3.png)

## Step 4: Connect the LED to GPIO 0 and GND

Connect the free end of the 220Î© resistor to GPIO 0.

# Full Project Code

Upload and run this code after the individual tests work correctly.

```python
from machine import Pin, PWM
import bluetooth
import time
from ble_uart import BLEUART

led = PWM(Pin(0))
led.freq(1000)

ble = bluetooth.BLE()
ble.active(True)
uart = BLEUART(ble, name='Pico-Lamp')

brightness_percent = 0

def set_brightness(percent):
    global brightness_percent
    if percent < 0:
        percent = 0
    if percent > 100:
        percent = 100
    brightness_percent = percent
    duty = int((percent * 65535) / 100)
    led.duty_u16(duty)


def on_rx(data):
    command = data.decode('utf-8').strip().lower()
    print('Received command:', command)

    if command == 'on':
        set_brightness(100)
        uart.write(b'Lamp ON at 100%\n')
    elif command == 'off':
        set_brightness(0)
        uart.write(b'Lamp OFF\n')
    elif command == 'low':
        set_brightness(25)
        uart.write(b'Lamp LOW at 25%\n')
    elif command == 'medium':
        set_brightness(60)
        uart.write(b'Lamp MEDIUM at 60%\n')
    elif command == 'high':
        set_brightness(100)
        uart.write(b'Lamp HIGH at 100%\n')
    elif command == 'status':
        uart.write(('Lamp brightness: {}%\n'.format(brightness_percent)).encode())
    elif command == 'help':
        uart.write(b'Commands: on, off, low, medium, high, status, help, or 0-100\n')
    else:
        try:
            percent = int(command)
            set_brightness(percent)
            uart.write(('Lamp brightness: {}%\n'.format(brightness_percent)).encode())
        except ValueError:
            uart.write(b'Unknown command. Send help.\n')

uart.on_rx(on_rx)
set_brightness(0)

print('Bluetooth smart lamp ready')
print('Send on, off, low, medium, high, status, help, or a number from 0 to 100')

while True:
    time.sleep(0.1)
```

# How the Code Works

| Code Section       | What It Does                                            | Why It Matters                                        |
| ------------------ | ------------------------------------------------------- | ----------------------------------------------------- |
| PWM LED setup      | Creates a PWM output on GPIO 0                          | PWM makes smooth brightness control possible          |
| `set_brightness()` | Converts a 0 to 100 percentage into a PWM duty value    | Students can use simple brightness numbers            |
| Named commands     | Provides easy preset commands such as low, medium, high | These are easier for beginners than only raw numbers  |
| Status reply       | Sends the current brightness back to the phone          | The student gets confirmation that the command worked |

# Troubleshooting

| Problem                     | Possible Cause                                               | Solution                                                                                               |
| --------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------ |
| LED does not light          | LED is reversed, the resistor is missing, or wiring is wrong | Check LED polarity, the resistor, and GPIO 0 wiring                                                    |
| Brightness does not change  | The command text is wrong or PWM test failed                 | Send `on`, `off`, `low`, `medium`, `high`, or a number from 0 to 100 and rerun the PWM lamp test       |
| Phone cannot find Pico-Lamp | BLE helper files are missing or Bluetooth is not active      | Check that `ble_uart.py` and `ble_advertising.py` are saved on the Pico and rerun the advertising test |
