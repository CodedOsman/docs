## Bluetooth Relay Switch

## Overview

Build a Bluetooth-controlled relay switch that can be turned on or off from a phone.

This project demonstrates how BLE control can switch a low-voltage external device through a relay module.

The final result should let your phone turn the relay on or off and receive a text reply showing the relay state.

Place the 1-channel relay module on the breadboard or beside it where the pins are easy to reach.

Identify VCC, GND, IN, COM, NO, and NC before wiring.

![Step 2](../../../assets/beginner/BLUETOOTH/BLUETOOTH%20RELAY%20SWITCH/circuit_3.png)

### Step 3: Connect Relay Power

Connect relay VCC to 5V / VSYS.

Connect relay GND to GND.

![Step 3](../../../assets/beginner/BLUETOOTH/BLUETOOTH%20RELAY%20SWITCH/circuit_4.png)

### Step 4: Connect the Relay IN Pin

Connect relay IN to GPIO 0.

This is the Bluetooth-controlled relay signal.

### Step 5: Optional Low-Voltage Load

If you connect a test load, use only a safe low-voltage load.

Wire the load through the relay output side according to the load's power supply.

Do not connect mains AC power in this beginner project.

## Full Project Code

Upload and run this code after the individual tests work correctly.

```python
from machine import Pin
import bluetooth
import time
from ble_uart import BLEUART

relay = Pin(0, Pin.OUT)
relay.value(1)

ble = bluetooth.BLE()
ble.active(True)
uart = BLEUART(ble, name='Pico-Relay')


def relay_state_text():
    return 'ON' if relay.value() == 0 else 'OFF'


def on_rx(data):
    command = data.decode('utf-8').strip().lower()
    print('Received command:', command)

    if command in ('1', 'on'):
        relay.value(0)
        uart.write(b'Relay ON\n')
    elif command in ('0', 'off'):
        relay.value(1)
        uart.write(b'Relay OFF\n')
    elif command == 'toggle':
        relay.value(0 if relay.value() else 1)
        uart.write(('Relay ' + relay_state_text() + '\n').encode())
    elif command == 'status':
        uart.write(('Relay status: ' + relay_state_text() + '\n').encode())
    elif command == 'help':
        uart.write(b'Commands: on, off, toggle, status, help\n')
    else:
        uart.write(b'Unknown command. Send help.\n')


uart.on_rx(on_rx)

print('Bluetooth relay controller ready')
print('Connect with a BLE app and send commands: on, off, toggle, status, help')

while True:
    time.sleep(0.1)
```

## How the Code Works

| Code Section           | What It Does                                      | Why It Matters                                        |
| ---------------------- | ------------------------------------------------- | ----------------------------------------------------- |
| Active-low relay setup | Starts the relay in the OFF state                 | This avoids turning on a load unexpectedly at startup |
| relay_state_text()     | Creates a readable ON or OFF message              | The phone app receives clear text feedback            |
| BLEUART service        | Provides a text-based Bluetooth link to the phone | This is the control channel for the relay             |
| on_rx() callback       | Processes incoming control commands               | This is where the relay state changes happen          |

## Troubleshooting

| Problem                      | Possible Cause                                          | Solution                                                                          |
| ---------------------------- | ------------------------------------------------------- | --------------------------------------------------------------------------------- |
| Relay does not click         | Wrong power wiring or active-low logic is misunderstood | Check VCC, GND, and remember many modules turn ON when the input goes low         |
| Phone cannot find Pico-Relay | BLE libraries are missing or Bluetooth is not active    | Check the library files and rerun the advertising test                            |
| Relay turns the wrong way    | The module logic is different from expected             | Test with the relay click test and adjust only if your module behaves differently |
