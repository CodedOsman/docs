# Project 1.11.2

## IoT Plant Status Monitor

# Overview

Build a plant-status dashboard that combines soil moisture, temperature, humidity, and light readings in one project.

This project demonstrates combining I2C and ADC sensors in a single monitor.

The final result should show plant-related status information on both an OLED display and a local web dashboard.

# Full Project Code

Upload and run this code after the individual tests work correctly.

```python
import network
import socket
import time
from machine import ADC, I2C, Pin
import sh1106
import BME280

SSID = 'YOUR_WIFI_NAME'
PASSWORD = 'YOUR_WIFI_PASSWORD'

i2c = I2C(0, sda=Pin(8), scl=Pin(9), freq=400000)
oled = sh1106.SH1106_I2C(128, 64, i2c)

try:
    bme = BME280.BME280(i2c=i2c, address=0x76)
except OSError:
    bme = BME280.BME280(i2c=i2c, address=0x77)

soil_adc = ADC(Pin(26))
light_adc = ADC(Pin(27))

SOIL_DRY = 58000
SOIL_WET = 20000


def soil_percent():
    raw = soil_adc.read_u16()
    if raw >= SOIL_DRY:
        return 0
    if raw <= SOIL_WET:
        return 100
    return int((SOIL_DRY - raw) / (SOIL_DRY - SOIL_WET) * 100)


def light_percent():
    raw = light_adc.read_u16()
    return int((raw / 65535) * 100)


def plant_status(soil, light):
    if soil < 25:
        return 'Needs Water'
    if light < 20:
        return 'Needs Light'
    return 'Plant OK'


def web_page(soil, temp, hum, light, status):
    return '''<!DOCTYPE html>
<html>
<head>
    <meta name='viewport' content='width=device-width, initial-scale=1'>
    <meta http-equiv='refresh' content='5'>
    <title>IoT Plant Status</title>
</head>
<body style='font-family:Arial;text-align:center;padding:30px'>
    <h1>IoT Plant Status</h1>
    <p>Soil Moisture: {}%</p>
    <p>Temperature: {}</p>
    <p>Humidity: {}</p>
    <p>Light: {}%</p>
    <h3>{}</h3>
</body>
</html>'''.format(soil, temp, hum, light, status)


wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

print('Connecting to Wi-Fi...')
for _ in range(15):
    if wlan.isconnected():
        break
    time.sleep(1)

if not wlan.isconnected():
    raise RuntimeError('Wi-Fi connection failed')

ip_address = wlan.ifconfig()[0]
print('Connected. Open http://{} in your browser'.format(ip_address))

address = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
server = socket.socket()
server.bind(address)
server.listen(1)
server.settimeout(0.2)

while True:
    soil = soil_percent()
    light = light_percent()
    temp = str(bme.temperature)
    hum = str(bme.humidity)
    status = plant_status(soil, light)

    oled.fill(0)
    oled.text('Plant Status', 12, 0, 1)
    oled.text('Soil:{}%'.format(soil), 0, 18, 1)
    oled.text('Light:{}%'.format(light), 0, 32, 1)
    oled.text(status[:16], 0, 50, 1)
    oled.show()

    try:
        client, client_address = server.accept()
    except OSError:
        continue

    client.recv(1024)
    response = web_page(soil, temp, hum, light, status)
    client.send('HTTP/1.1 200 OK\r\nContent-Type: text/html\r\nConnection: close\r\n\r\n'.encode())
    client.sendall(response.encode())
    client.close()
```

# How the Code Works

| Code Section | What It Does | Why It Matters |
| --- | --- | --- |
| Mixed sensor setup | Uses one I2C bus for BME280 and OLED plus two ADC inputs for soil and light | This project brings several beginner sensor ideas together |
| soil_percent() | Converts the moisture sensor reading into a percentage | The plant dashboard needs a simple moisture value |
| light_percent() | Converts the LDR reading into a light percentage | This lets the project describe the room light level simply |
| plant_status() | Creates a simple overall plant message | The dashboard becomes easier to understand than showing only raw numbers |

# Expected Result

After entering your Wi-Fi details and running the code, the OLED and browser page should show soil moisture, temperature, humidity, light level, and a simple overall plant status such as Needs Water, Needs Light, or Plant OK.

# Troubleshooting

| Problem | Possible Cause | Solution |
| --- | --- | --- |
| Soil reading looks wrong | Moisture calibration values do not match your sensor | Print raw ADC values and adjust SOIL_DRY and SOIL_WET |
| Light reading never changes | LDR voltage divider is wired incorrectly | Check the 3.3V -> LDR -> GPIO 27 junction -> 10kÎ© -> GND path |
| No BME280 values | Wrong I2C wiring or wrong address | Run the I2C scanner and try 0x76 or 0x77 |
| OLED shows nothing | Wrong pins or missing sh1106.py | Check GPIO 8/9 and confirm the library file is on the Pico |
