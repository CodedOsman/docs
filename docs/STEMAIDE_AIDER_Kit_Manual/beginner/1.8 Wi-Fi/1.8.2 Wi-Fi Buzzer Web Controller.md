# Full Project Code

```python
import network
import socket
import time
from machine import Pin

SSID = 'YOUR_WIFI_NAME'
PASSWORD = 'YOUR_WIFI_PASSWORD'

buzzer = Pin(0, Pin.OUT)

def buzz_once():
    buzzer.on()
    time.sleep(0.3)
    buzzer.off()

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

print('Connecting to Wi-Fi...')

for _ in range(15):
    if wlan.isconnected():
        break
    time.sleep(1)

if not wlan.isconnected():
    raise RuntimeError('Wi-Fi connection failed')

ip_address = wlan.ifconfig()[0]
print('Connected. Open http://{} in your browser'.format(ip_address))

html = """<!DOCTYPE html>
<html>
<head>
<meta name='viewport' content='width=device-width, initial-scale=1'>
<title>Wi-Fi Buzzer</title>
</head>
<body style='font-family:Arial;text-align:center;padding:40px'>
<h1>Wi-Fi Buzzer Control</h1>
<p>Press the button to sound the buzzer.</p>
<a href='/beep'>
<button style='padding:16px 28px;font-size:20px'>BEEP</button>
</a>
</body>
</html>"""

address = socket.getaddrinfo('0.0.0.0', 80)[0][-1]

server = socket.socket()
server.bind(address)
server.listen(1)

while True:

    client, client_address = server.accept()
    print('Client connected from', client_address)

    request = client.recv(1024).decode()

    if 'GET /beep' in request:
        buzz_once()
        print('Buzzer sounded')

    client.send(
        'HTTP/1.1 200 OK\r\n'
        'Content-Type: text/html\r\n'
        'Connection: close\r\n\r\n'
    )

    client.sendall(html)
    client.close()
```

# How the Code Works

| Code Section  | What It Does                                      | Why It Matters                                           |
| ------------- | ------------------------------------------------- | -------------------------------------------------------- |
| `buzz_once()` | Turns the buzzer on for a short time and then off | Keeps the sound easy to control                          |
| Wi-Fi setup   | Connects the Pico 2 W to the network              | The web server needs Wi-Fi before the page can be opened |
| HTML page     | Creates a simple page with one BEEP button        | The browser needs this page to send the command          |
| Request check | Looks for `/beep` in the browser request          | Tells the Pico when to sound the buzzer                  |

# Troubleshooting

| Problem                                | Possible Cause                                                 | Solution                                                           |
| -------------------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------ |
| No sound                               | Buzzer wiring is wrong or buzzer type is unsuitable            | Check polarity and confirm you are using an active buzzer          |
| Wi-Fi never connects                   | Wrong network details or unsupported network                   | Recheck the SSID and password and make sure the network is 2.4 GHz |
| Web page opens but button does nothing | Browser request is not reaching the Pico or buzzer test failed | Run the buzzer test first and then recheck the request code        |
