# Full Project Code

```python
import network
import socket
import time
from machine import Pin

SSID = 'YOUR_WIFI_NAME'
PASSWORD = 'YOUR_WIFI_PASSWORD'

pir = Pin(1, Pin.IN)

last_motion = 0
motion_count = 0
last_motion_time = -10000

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

print('Connecting to Wi-Fi...')

for _ in range(15):
    if wlan.isconnected():
        break
    time.sleep(1)

if not wlan.isconnected():
    raise RuntimeError('Wi-Fi connection failed')

ip_address = wlan.ifconfig()[0]

print('Connected. Open http://{} in your browser'.format(ip_address))
print('Wait 15 seconds for PIR warm-up')

time.sleep(15)

def web_page(recent_motion, count):

    status_text = 'MOTION DETECTED!' if recent_motion else 'No Motion'
    color = '#ff5555' if recent_motion else '#44aa44'

    return """<!DOCTYPE html>
<html>
<head>
<meta name='viewport' content='width=device-width, initial-scale=1'>
<meta http-equiv='refresh' content='2'>
<title>PIR Motion Alert</title>
</head>
<body style='font-family:Arial;text-align:center;padding:40px'>
<h1>PIR Motion Alert</h1>
<div style='font-size:32px;color:STATUS_COLOR'>STATUS_TEXT</div>
<p>Total detections: COUNT_TEXT</p>
<p>Page refreshes every 2 seconds</p>
</body>
</html>""".replace(
        'STATUS_TEXT', status_text
    ).replace(
        'STATUS_COLOR', color
    ).replace(
        'COUNT_TEXT', str(count)
    )

address = socket.getaddrinfo('0.0.0.0', 80)[0][-1]

server = socket.socket()
server.bind(address)
server.listen(1)
server.settimeout(0.2)

while True:

    motion = pir.value()

    if motion == 1 and last_motion == 0:
        motion_count += 1
        last_motion_time = time.ticks_ms()
        print('Motion detected! Count:', motion_count)

    last_motion = motion

    recent_motion = (
        time.ticks_diff(
            time.ticks_ms(),
            last_motion_time
        ) < 5000
    )

    try:
        client, client_address = server.accept()
    except OSError:
        continue

    print('Client connected from', client_address)

    client.recv(1024)

    response = web_page(recent_motion, motion_count)

    client.send(
        'HTTP/1.1 200 OK\r\n'
        'Content-Type: text/html\r\n'
        'Connection: close\r\n\r\n'.encode()
    )

    client.sendall(response.encode())
    client.close()
```

# How the Code Works

| Code Section       | What It Does                                                            | Why It Matters                                                |
| ------------------ | ----------------------------------------------------------------------- | ------------------------------------------------------------- |
| PIR Input          | Reads the motion sensor state                                           | Provides the motion event signal                              |
| Edge Detection     | Counts only a new motion event when the signal changes from LOW to HIGH | Prevents one long motion signal from being counted repeatedly |
| `last_motion_time` | Stores when the most recent motion occurred                             | Allows the page to show a recent alert                        |
| Web Page Response  | Shows current alert status and total detection count                    | Creates a simple Wi-Fi dashboard                              |

# Troubleshooting

| Problem                     | Possible Cause                                                    | Solution                                               |
| --------------------------- | ----------------------------------------------------------------- | ------------------------------------------------------ |
| No motion is detected       | PIR still warming up or incorrect wiring                          | Wait longer and recheck the OUT pin on GPIO 1          |
| False triggers happen often | Sensor sensitivity is too high or the area contains heat movement | Reposition the PIR and reduce sensitivity              |
| Web page never changes      | Motion count is not updating or browser is not refreshing         | Watch the Shell output and confirm the count increases |
