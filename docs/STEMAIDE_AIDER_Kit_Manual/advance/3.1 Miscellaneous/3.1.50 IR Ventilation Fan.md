## Overview

The Pico runs a relay-driven fan when an IR occupancy sensor sees a person and keeps it running briefly after the person leaves.

Short ventilation run-on time can improve air quality without leaving the fan on constantly.

A Pico 2 W prototype with an IR occupancy sensor, relay output, fan load, and LED indicator.

Presence-based control, run-on timing, and safe relay switching for a fan system.

## Circuit Connections

|---------------|-------------|---------------------------------|-------|
| IR sensor VCC | 3.3V output | Physical pin 36 | Check that the module supports 3.3V |
| IR sensor GND | GND | Physical pin 38 | Common ground |
| IR sensor OUT | GPIO 14 | GPIO 14 / physical pin 19 | Active-low occupancy input |
| Relay IN | GPIO 15 | GPIO 15 / physical pin 20 | Relay control |
| Relay VCC | External 5V or module-rated supply | Not a Pico GPIO pin | Use the correct module supply |
| Relay GND | GND | Physical pin 38 | Share ground unless fully isolated |
| LED anode | 220 ohm resistor to GPIO 16 | GPIO 16 / physical pin 21 | Fan state LED |
| LED cathode | GND | Physical pin 23 or 38 | Ground return |

---

## Wiring Diagram

```
  3.3V -> IR sensor VCC
  GND -> IR sensor GND, relay GND, LED cathode
  IR sensor OUT -> GPIO 14
  GPIO 15 -> relay IN
  GPIO 16 -> 220 ohm resistor -> LED anode
  External supply -> relay-switched fan
```

## Step-by-Step Assembly

1. Connect the IR sensor to Pico 3.3V, GND, and GPIO 14.
2. Connect the relay input to GPIO 15 and power the relay module correctly.
3. Wire the LED to GPIO 16 through a 220 ohm resistor and connect the LED cathode to GND.
4. Dry-test the relay before attaching the real fan load.
5. Connect the fan to its external power supply only after the occupancy logic is validated.

3. **Test the output device**: Toggle the relay and LED from a short test script before attaching the fan.

## Full Project Code

After completing and checking the circuit connections, open Thonny IDE, copy and paste this code into a new file or upload the project file to the Raspberry Pi Pico 2 W, then run it from Thonny.

```python
from machine import Pin
import time

IR_PIN = 14
RELAY_PIN = 15
LED_PIN = 16
RUN_ON_MS = 30000

ir = Pin(IR_PIN, Pin.IN, Pin.PULL_UP)
relay = Pin(RELAY_PIN, Pin.OUT)
led = Pin(LED_PIN, Pin.OUT)
last_seen_ms = None

relay.off()
led.off()
print("IR ventilation fan ready")

while True:
    now = time.ticks_ms()
    occupied = ir.value() == 0

    if occupied:
        last_seen_ms = now

    fan_on = last_seen_ms is not None and time.ticks_diff(now, last_seen_ms) <= RUN_ON_MS
    relay.value(1 if fan_on else 0)
    led.value(1 if fan_on else 0)

    mode = "OCCUPIED" if occupied else ("RUN_ON" if fan_on else "IDLE")
    print("Mode:", mode)
    time.sleep(1)
```

### Validation Checks

- **Normal condition test**: confirm the system stays in its safe or idle state under baseline conditions
- **Warning condition test**: move the input close to the chosen threshold and confirm the transition is sensible
- **Critical condition test**: trigger the strongest alert or control state and confirm the output response is correct
- **Calibration test**: compare the sensor or timing against a known reference or repeated trial
- **Limitation test**: deliberately create an awkward or noisy condition and note how the prototype behaves

### Deployment and Limitations

- This prototype could be used in school, farm, or sanitation demonstrations with safe low-voltage loads
- Before deployment, the load wiring, enclosure, and power design need careful review
- Actuator systems need maintenance planning because relay wear and sensor drift affect reliability

## Troubleshooting

| Problem | Possible Cause | Solution |
|---------|----------------|----------|
| Fan turns off too quickly | Run-on timer is too short | Increase RUN_ON_MS and retest the exit timing |
| Fan stays on all the time | Sensor sees constant occupancy or reflective noise | Reposition the sensor and check whether the output idles high |
| Relay works but fan does not spin | External power or load wiring is wrong | Test the fan separately from the relay and verify the supply voltage |
| Sensor misses people | Detection zone is too narrow | Aim the IR sensor more directly at the walkway or doorway |
