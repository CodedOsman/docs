# Intermediate Projects

Build on your foundation by combining multiple sensors, inputs, and outputs in intermediate projects! This section integrates the components you learned individually to create more complex and interactive systems.

## Components Used in Intermediate Projects

Intermediate projects combine components from beginner projects in meaningful ways:

- **Push Button** — Digital input switch for user control and triggering actions.
- **Buzzer** — Audio output for alerts and feedback signals.
- **RGB Module** — Multicolor LED display for visual feedback and status indication.
- **Traffic Light Module** — RGB simulation for sequence and state management.
- **LDR Module** — Light sensor for detecting environmental conditions and triggering responses.
- **Ultrasonic Sensor** — Distance measurement for proximity detection and automation.
- **Servo Motor** — Precision movement for mechanical control and positioning.
- **Sound Sensor** — Audio detection for sound-triggered applications and threshold-based alarms.
- **LED (Basic Output)** — Single or multiple LEDs for indicators and status displays in alarm systems.

## Available Projects

These projects combine two or more components to create intelligent systems:

- **2.1 Button and Buzzer** — Button-controlled sound alerts and feedback.
- **2.2 Button and RGB** — Button-triggered color changes and light sequences.
- **2.3 Button and Traffic Light** — Button-controlled traffic light sequences.
- **2.4 LDR and RGB** — Light-sensitive color-changing displays.
- **2.5 LDR and Traffic Light** — Ambient light-responsive traffic light control.
- **2.6 Ultrasonic and Buzzer** — Distance-triggered alarm systems.
- **2.7 Ultrasonic and RGB** — Distance-based visual feedback with color gradients.
- **2.8 Sound Sensor and Traffic Light** — Sound-activated traffic light sequences.
- **2.9 Servo and Ultrasonic** — Distance-triggered servo movement for gating systems.
- **2.10 Smart Alarm Projects** — Advanced multi-sensor alarm systems with threshold-based triggering.
- **2.11 Potentiometer Buzzer** — Create a distance theremin using a potentiometer and buzzer.
- **2.12 Automatic traffic detention** — Proximity-based smart street lighting system.
- **2.13 Intruder Alarm System** — Ultrasonic-triggered home security alarm system.
