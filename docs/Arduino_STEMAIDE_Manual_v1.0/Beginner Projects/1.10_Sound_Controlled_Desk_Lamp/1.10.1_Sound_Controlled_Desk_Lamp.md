# Project 1.10.0: Sound-Controlled Desk Lamp

| **Description** | This project creates a desk lamp that can be controlled using claps or loud sounds. The sound sensor toggles the lamp modes, while a potentiometer adjusts the brightness and colour intensity of the RGB LED. |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use case**    | Hands-free lighting, smart desk lamps, bedroom lighting, interactive electronics.                                                                                                                                 |

## Building the circuit

Things Needed:

- Arduino Uno = 1
- Arduino USB cable = 1
- Breadboard = 1
- Sound Sensor Module = 1
- RGB LED = 1
- Potentiometer = 1
- Jumper Wires = multiple

## WIRING THE CIRCUIT

**Step 1:** Connect one jumper wire from the 5V pin on the Arduino Uno to the positive rail of the breadboard and connect another jumper wire from the GND pin on the Arduino Uno to the negative rail of the breadboard.


**Step 2:** Place the SOUND SENSOR on the breadboard.
Connect the sound sensor:
- VCC → 5V
- GND → GND
- DO → Pin 7


**Step 3:** Place the RGB LED on the breadboard.
Connect the RGB LED:
- Red → Pin 3
- Green → Pin 5
- Blue → Pin 6


**Step 4:** Place the POTENTIOMETER on the breadboard.
Connect the potentiometer:
- Left pin → 5V
- Right pin → GND
- Middle pin → A0


**Step 5:** After completing the wiring, connect the Arduino Uno to the computer using the USB cable.

## PROGRAMMING

Below is the Arduino code to make the Sound-Controlled Desk Lamp work.

**Step 1:** Open your Arduino IDE. See how to set up here: [Getting Started](../../Getting Started/Arduino_IDE_Setup.md).

**Step 2:** Copy and paste the following code into a new sketch:

```cpp
const int soundPin = 7;
const int redPin = 3;
const int greenPin = 5;
const int bluePin = 6;
const int potPin = A0;

int mode = 0; // 0=Red, 1=Green, 2=Blue, 3=White
bool lastSoundState = HIGH; 

void setup() {
  pinMode(soundPin, INPUT);
  pinMode(redPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int soundState = digitalRead(soundPin);
  
  // Detect a clap (transition from HIGH to LOW on most sound modules)
  if (soundState == LOW && lastSoundState == HIGH) {
    mode++;
    if (mode > 3) mode = 0; // cycle through 4 color modes
    delay(200); // debounce delay to prevent multiple triggers from one clap
  }
  lastSoundState = soundState;

  // Read the potentiometer to control brightness
  int potValue = analogRead(potPin);
  int brightness = map(potValue, 0, 1023, 0, 255);

  // Apply the selected mode and brightness
  if (mode == 0) {
    analogWrite(redPin, brightness);
    analogWrite(greenPin, 0);
    analogWrite(bluePin, 0);
  } else if (mode == 1) {
    analogWrite(redPin, 0);
    analogWrite(greenPin, brightness);
    analogWrite(bluePin, 0);
  } else if (mode == 2) {
    analogWrite(redPin, 0);
    analogWrite(greenPin, 0);
    analogWrite(bluePin, brightness);
  } else if (mode == 3) {
    analogWrite(redPin, brightness);
    analogWrite(greenPin, brightness);
    analogWrite(bluePin, brightness);
  }
}
```

**Step 3:** Save your code. _See the [Getting Started](../../Getting Started/Arduino_IDE_Setup.md) section_.

**Step 4:** Select Arduino Uno from Tools → Board.

**Step 5:** Select the correct COM port from Tools → Port.

**Step 6:** Click Upload.

### EXPLANATION
The sound sensor detects claps and cycles through different lamp modes (colors). The potentiometer controls brightness, allowing the RGB LED to display different colours at adjustable intensity levels.

## OBSERVATION

- Each clap changes the lamp mode.
- The potentiometer adjusts brightness.
- The RGB LED changes between multiple colour presets.

## CONCLUSION

This project demonstrates sound sensing, PWM control, RGB colour mixing, and analog input processing.
